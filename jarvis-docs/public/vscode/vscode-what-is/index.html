Что такое VS Code Tunnel и зачем он нужен

3 мин · обновлено 2026-05-10
Окно в офис Агента - видите его рабочий стол из любого браузера.

Telegram - главный способ общения с Агентом. Но иногда хочется заглянуть "за кулисы": посмотреть файлы на сервере, отредактировать код, проверить что Агент написал. Для этого есть VS Code Tunnel.

Что это

VS Code - это редактор кода, который работает прямо в браузере. Tunnel (туннель) - безопасное соединение с сервером Вашего Агента. Вместе они дают Вам доступ к файлам Агента через браузер, как если бы Вы сидели за его компьютером.

Простым языком

Telegram - это как переписка с сотрудником. VS Code Tunnel - это как зайти к нему в кабинет и посмотреть на его монитор.

Зачем это нужно

Посмотреть результаты работы. Агент сделал лендинг, написал код или создал таблицу - Вы открываете VS Code и видите все файлы, можете их скачать или отредактировать.

Быстро поправить текст. Вместо "поменяй в файле X слово Y на Z" - открываете файл, правите сами, сохраняете. Иногда это быстрее, чем объяснять.

Работать с кодом. Если Вы разбираетесь в коде - можете писать код параллельно с Агентом. Он видит Ваши изменения, Вы видите его.

Проверить память и настройки. Открыть MEMORY.md, SOUL.md, GOALS.md и посмотреть, что там записано. Или поправить прямо руками.

Нужен ли VS Code всем?

Нет. Если Вам достаточно общения через Telegram - VS Code можно не трогать. Всё, что умеет VS Code, Агент умеет делать сам по Вашей просьбе.

VS Code полезен, если:

  • Вы хотите видеть файлы глазами
  • Вы пишете код сами
  • Вы хотите тонко настроить Агента (SOUL.md, скиллы)
  • Вам нравится иметь полный контроль

Как выглядит

Вы открываете vscode.dev в браузере, подключаетесь к серверу Агента - и видите дерево файлов, терминал, редактор. Всё как в обычном редакторе кода, но файлы - на сервере Агента.

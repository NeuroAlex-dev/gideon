Эволюция Агента — как Ваш Агент становится умнее

3 мин · обновлено 2026-05-10
Конструктор Агентов растёт каждую неделю — Ваш Агент растёт вместе с ним.

Конструктор Агентов развивается. Появляются новые скиллы, улучшается движок бота, выходят новые интеграции. И всё это можно добавить к Вашему Агенту буквально одной командой.

Главный смысл

Главный смысл

Я обновляю Конструктор Агентов — Вы устанавливаете обновление одной командой /new. Ваш Агент становится умнее без переустановки и без потери данных.

Как работает обновление

Когда выходит новая версия:

  1. @jarchitect_bot пишет Вам: «Доступно обновление»
  2. Вы отправляете команду /new в @jarchitect_bot
  3. Бот присылает одну команду установки (curl | bash)
  4. Вы открываете терминал на сервере (через VS Code Tunnel)
  5. Вставляете команду и запускаете — 1-2 минуты
  6. Агент перезапускается с новыми возможностями

Что обновляется

📡 @jarchitect_bot /new
       │
       ↓ команда curl | bash
       │
   ┌───┴───┬─────────────────┬──────────────────┐
   │       │                 │                  │
   🛠 Скиллы     📋 Шаблоны   🤖 Движок бота   ⭐ Новые фишки

Скиллы. Это специалисты в команде Вашего Агента. Появился youtube-scenario — Агент умеет писать сценарии. Новый report-builder — готовит отчёты по шаблону. После /new все новые скиллы сразу работают.

Шаблоны (CLAUDE.md, SERVICES.md и другие технические файлы). Улучшаются, чтобы Агент работал быстрее и умнее. Как обновление прошивки телефона — телефон становится лучше, Ваши данные остаются.

Движок бота (~/.agent/bot/index.js). Новые команды, ускорения, улучшения. Перед каждым обновлением делается бэкап — можно откатиться.

Что не меняется никогда

Никакое обновление не трогает Вашу личность, память и проекты. SOUL.md который Вы написали под себя, MEMORY.md который Агент копил месяцами, проекты в ~/projects/ — это Ваше навсегда.

Движок становится мощнее — душа Агента остаётся Ваша.

Мой Вам совет

Когда увидите уведомление об обновлении — запускайте. Память не потеряется, личность не сбросится, проекты не пропадут. Бэкап делается автоматически перед каждой установкой.

Как добавить API-ключ через /settings

2 мин · обновлено 2026-05-09
Ключи в сейф, а не на холодильник.

API-ключи никогда не пишите в чат текстом. Только через меню /settings.

Шаги

  1. В Telegram: /settings → 🔑 Переменные окружения
  2. Выбираете нужный сервис (или «Добавить свою»)
  3. Копируете ключ из консоли сервиса
  4. Присылаете боту — он моментально удаляет ваше сообщение и сохраняет ключ

Готовые пресеты

  • Deepgram (голос), GitHub, Vercel, OpenRouter, Notion, OpenAI

Что важно

Если случайно отправили ключ в обычный чат — он попадёт в логи.

Сразу отзовите ключ в консоли сервиса и выпустите новый.

Расширение Claude Code

VS Code установлен. Теперь добавим расширение, которое превратит обычный редактор в рабочее место с ИИ-помощником. Расширение Claude Code встраивает панель чата прямо в интерфейс редактора.

Аналогия

Расширение — как приложение на телефоне. Сам телефон работает и без них, но без нужных приложений толку мало. VS Code — это телефон. Claude Code — приложение, которое делает его по-настоящему полезным.

Шаг 1: Открыть магазин расширений

Откройте VS Code. В левой панели найдите иконку из четырёх квадратиков — это Extensions (магазин расширений). Нажмите на неё.

Совет

Горячая клавиша: Cmd + Shift + X на Mac, Ctrl + Shift + X на Windows. Запомните — пригодится не раз.

Откроется панель со строкой поиска и списком популярных расширений.

Шаг 2: Найти и установить

В поле поиска введите Claude Code. В результатах найдите расширение от Anthropic — это официальный разработчик Claude. Нажмите кнопку Install.

Установка занимает несколько секунд. После этого в левой панели появится иконка Claude.

Внимание

Убедитесь, что автор расширения — именно Anthropic. В магазине встречаются неофициальные копии. Устанавливайте только оригинал.

Шаг 3: Авторизация

Для работы нужно войти в аккаунт Anthropic. Откройте панель Claude Code одним из двух способов.

Способ 1 — Command Palette (рекомендуется): Нажмите Cmd + Shift + P (Mac) или Ctrl + Shift + P (Windows). В появившейся строке введите Claude Code: Open in Side Bar и нажмите Enter.

Способ 2 — иконка в панели: Нажмите на иконку Claude (оранжевый логотип) в левой боковой панели.

В открывшейся панели нажмите Sign In. Откроется браузер со страницей авторизации Anthropic. Войдите в свой аккаунт, подтвердите доступ и вернитесь в VS Code.

Внимание

Для регистрации и авторизации потребуется подключение через зарубежный сервер и зарубежная сим-карта. Настройте доступ заранее, чтобы не прерывать процесс. Детали уточняйте у ассистента курса.

Совет

Если иконка Claude пропала из боковой панели — используйте Command Palette. Горячая клавиша: Cmd + Shift + P (Mac) или Ctrl + Shift + P (Windows).

Шаг 4: Проверка

Откройте панель Claude Code. В поле ввода напишите «Привет» и нажмите Enter. Если Claude ответил — всё настроено правильно.

Контрольный чек-лист:

  • Иконка Claude появилась в боковой панели
  • Панель чата открывается без ошибок
  • Вы авторизованы (нет кнопки Sign In)
  • Claude отвечает на сообщения

Все четыре пункта выполнены — вы готовы к работе.

Если что-то не работает

Claude не отвечает на сообщения. Проверьте подключение к интернету. Убедитесь, что подписка Anthropic активна. Перезапустите VS Code. Если не помогло — удалите расширение и установите заново.

Кнопка Sign In не появляется. Закройте VS Code и откройте снова. Проверьте подключение через зарубежный сервер. Попробуйте открыть панель через Command Palette.

Браузер открылся, но авторизация не проходит. Проверьте, что вы входите в правильный аккаунт Anthropic. Попробуйте другой браузер. Очистите кэш текущего браузера и повторите попытку.

Информация

Не паникуйте. Любая из этих проблем решается за пару минут. Если ничего не помогает — напишите в чат поддержки курса, разберёмся вместе.

Что дальше

Расширение установлено, авторизация пройдена. Пора создать первый проект и дать Claude Code настоящую задачу.

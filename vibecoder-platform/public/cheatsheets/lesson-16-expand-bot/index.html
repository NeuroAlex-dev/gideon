Как работает бот и почему он «пропадает»

Бот запускается с вашего компьютера

Claude Code запускает программу в терминале — и пока она работает, бот отвечает. Перезапустили компьютер или закрыли терминал — бот перестал работать. Нужно снова попросить Claude Code: «Запусти бота».

ПроблемаРешение
Локальная установка — работает, пока включён компьютер и запущена программа. После перезагрузки — запускать зановоУдалённый сервер — чтобы бот работал 24/7, перенесите его на VPS. Спросите ИИ: «Как установить бота на удалённый сервер?»

3 способа мониторинга трендов

1. Perplexity через OpenRouter — агент TrendScout

TrendScout-агент выходит в интернет через Perplexity и ищет актуальные посты, новости и темы по вашей нише. Уже встроен в бота с прошлого урока. Подключается через OpenRouter — ничего отдельно устанавливать не нужно.

Стоимость: ~$0.03 за поиск

2. Apify AgentSkills — анализ соцсетей

Скилл от Apify для мониторинга соцсетей. При регистрации дают $5 бесплатно (~10–15 исследований). Когда закончится — зарегистрировать новый аккаунт и поменять токен.

Стоимость: $5 бесплатно при регистрации, потом платный

3. Скилл из skills.sh — Google Trends + Reddit

Скилл Social Media Trends с skills.sh. Даёт доступ к Google Trends (тренды поисковых запросов) и Reddit Scraper (реальные обсуждения на форуме). Оба источника бесплатные. Нужно только установить Python-зависимости.

Стоимость: Бесплатно

Что такое скиллы и как их устанавливать

Скиллы — дополнительные навыки

Скиллы устанавливаются в Claude Code и расширяют его возможности. Как в «Матрице» — закачал скилл, и программа умеет что-то новое. Работают во всех редакторах: Claude Code, Cursor, Codex.

МаркетплейсОписание
skills.shКоллекция проверенных скиллов от Versailles. Безопасно. Ищите через поиск по теме
Apify AgentSkillsБольшой маркетплейс от Apify. Скиллы для работы с данными, соцсетями, анализа аудитории, трендов

Как установить скилл — 3 шага

  1. Найдите нужный скилл на маркетплейсе. Зайдите на skills.sh или Apify AgentSkills. Поищите по теме — например, «social media trends», «audience analysis»

  2. Скопируйте ссылку и отправьте Claude. Скопируйте URL страницы скилла. Вставьте в Claude Code: «Установи, пожалуйста, этот скилл: [ссылка]»

  3. Добавьте API-токен если нужен. Для Apify скиллов нужен токен Apify — добавить в .env. Для бесплатных скиллов из skills.sh токен не нужен. Если не хватает зависимостей — Claude сам их установит

Ключевые промпты урока

Промпт — Генерация картинки к посту

Добавь кнопку «Картинка» под готовым сгенерированным постом.
При нажатии — генерировать изображение, подходящее по смыслу поста.
Формат картинки: 16:9.
Модель генерации: flux-pro/v1.1-ultra (через OpenRouter) или аналог с минимальной стоимостью.
Подключай через OpenRouter.
Протестируй работу.

Промпт — Мониторинг трендов через Perplexity

Добавь функцию автопоиска трендов:
- Кнопка «Тренды» в главном меню
- Подменю: «Поиск по нише» и «Поиск по конкретной теме»
- Поиск выполняется через Perplexity (TrendScout-агент) через OpenRouter
- Результат: 3 трендовых топика для написания поста
- По каждой теме — кнопки «Написать пост 1», «Написать пост 2», «Написать пост 3»

Промпт — Установка скилла и интеграция в бота

Установи, пожалуйста, этот скилл: [ВСТАВЬТЕ ССЫЛКУ НА СКИЛЛ]

После установки:
- Проанализируй текущую архитектуру бота
- Дай рекомендации, как этот скилл усилит нашего бота
- Добавь новую кнопку в раздел «Тренды», которая использует этот скилл
- Не удаляй существующие источники поиска — только дополняй
- Протестируй работу

Промпт — Защита бота по Telegram ID

Ограничь доступ к боту только для моего Telegram ID: [ВСТАВЬТЕ ВАШ ID]
Все остальные пользователи должны получать сообщение об отсутствии доступа.
Это защитит от несанкционированного использования и перерасхода токенов.
Как найти свой Telegram ID

Откройте Telegram → поиск → введите Get ID → откройте бот @userinfobot или @getidsbot → нажмите Start. Бот сразу покажет ваш числовой ID.

Алгоритм когда что-то не работает

  1. Получили ошибку в боте. Сделайте скриншот ошибки (или скопируйте текст)

  2. Отправить в Claude Code. Вставьте скриншот или текст ошибки: «Вот ошибка, разберись и почини»

  3. Перезапустить бота. Попросите Claude: «Перезапусти бота». Или нажмите /start в боте после того, как Claude закончит правки

  4. Если не решается за 2–3 итерации. Скажите Claude: «Сам тестируй, если не работает — перезапускай, пока не заработает». Делегируйте отладку агенту

Лайфхаки урока

  • Расширяйте бота скиллами постепенно. Не удаляйте работающие источники — только добавляйте новые. Если новый скилл хуже — просто уберите кнопку, старый останется

  • Картинки для постов через OpenRouter. Flux Nano — самый дешёвый вариант. Добавьте кнопку под постом — и получаете визуал прямо в Telegram без переключения инструментов

  • Закройте бота по Telegram ID сразу. Иначе любой, кто найдёт вашего бота, может использовать ваши токены. Особенно важно если бот подключён к платным API

  • Apify закончился — регистрируйте снова. $5 бесплатно за регистрацию. Когда лимит кончится — новый аккаунт → новый токен → поменять в .env

  • Делегируйте тестирование агенту. «Тестируй сам, если не работает — перезапускай пока не заработает». Не тратьте своё время на итерации

  • Форматирование текста важнее содержания. Если пост с лишними символами (***, ##) — скажите Claude: «Убери форматирование Markdown, текст должен быть чистым для Telegram»

Фриланс на создании сайтов

Хотите зарабатывать на сайтах, но не хотите учиться программированию? Раньше создание лендинга занимало 2-3 недели. Теперь — 2-3 часа. Один проект приносит 30-80 тысяч рублей.

Что понадобится

  • Lovable — AI-помощник, который создаёт сайты по описанию
  • Портфолио — 3 демо-сайта для показа клиентам (создадим ниже)
  • Площадка для поиска клиентов — Telegram, Авито, сарафанное радио

Курсы по HTML и CSS не нужны. Дизайнер не нужен. Только вы и Lovable.

Аналогия

Вы — как повар с кухонным комбайном. Блюда по-прежнему ваши. Но вместо часа нарезки — 30 секунд. Клиенту важен результат, а не ваш способ готовки.

Шаг 1: Подготовка — выберите нишу

Не берите всех подряд. Выберите 2-3 типа клиентов.

Самые благодарные ниши для старта:

НишаСредний чекПочему легко
Стоматологии40-60К₽Типовая структура, много клиник
Салоны красоты30-50К₽Нужны фото + запись онлайн
Частные мастера15-30К₽Простой лендинг, быстрый результат
Мини-кафе и доставка30-50К₽Меню + контакты + карта
Совет

Начните с ниши, которую понимаете. Ходите к стоматологу — делайте сайты для стоматологов.

Что подготовить: определите 2-3 ниши, изучите сайты конкурентов, запишите типовую структуру блоков.

Шаг 2: Создание портфолио за один вечер

Клиенты не верят на слово. Им нужны примеры. Создадим 3 демо-сайта.

Откройте lovable.dev, создайте проект и вставьте промпт:

Создай лендинг для стоматологической клиники "Белая улыбка".

Структура:
1. Hero: название, слоган "Здоровые зубы — красивая улыбка",
   кнопка "Записаться на приём"
2. Блок услуг: 6 карточек (лечение, протезирование, чистка,
   отбеливание, имплантация, детская стоматология)
3. Блок "Почему мы": 4 преимущества с иконками
4. Отзывы: 4 карточки с цитатами пациентов
5. Форма записи: имя, телефон, выбор услуги
6. Контакты: адрес, телефон, время работы
7. Footer: логотип, ссылки, соцсети

Дизайн: светлый, голубой + белый, современный.
Tailwind CSS, shadcn/ui. Адаптивный макет.

Повторите ещё 2 раза для других ниш. За вечер у вас 3 сайта-примера.

Важно

Это демо-сайт. Не тратьте время на идеальные тексты. Главное — показать структуру и дизайн.

Ожидаемый результат: три работающих демо-лендинга с живыми ссылками.

Шаг 3: Поиск клиентов

Бесплатные каналы:

  1. Авито — объявление «Сделаю лендинг за 3 дня»
  2. Telegram-чаты вашего города — предложите услугу
  3. Сарафанное радио — расскажите друзьям и знакомым
  4. Прямой обход — зайдите в 10 заведений без сайта

Шаблон первого сообщения клиенту:

Здравствуйте! Увидел, что у [НАЗВАНИЕ_БИЗНЕСА]
нет сайта (или сайт устарел).

Я делаю современные лендинги для [НИША] за 3-5 дней.
Вот пример: [ССЫЛКА_НА_ДЕМО]

Стоимость — [ЦЕНА] руб. Включает дизайн, адаптацию
под телефоны, форму заявки и публикацию.

Интересно? Могу показать подробнее.
Про цены

Первым 2-3 клиентам сделайте скидку 50%. Ваша цель — отзывы и кейсы.

Ожидаемый результат: первый заказ. Клиент прислал информацию о бизнесе.

Шаг 4: Рабочий процесс — от брифа до сдачи

Этап 1: Бриф (30 минут). Узнайте у клиента: название, услуги, цвета, фото, контакты, примеры сайтов которые нравятся.

Этап 2: Создание в Lovable (1-2 часа). Вставьте данные клиента в шаблон:

Создай лендинг для [ТИП_БИЗНЕСА] "[НАЗВАНИЕ]" на React.

Структура:
1. Hero: "[НАЗВАНИЕ]", слоган "[СЛОГАН]",
   кнопка "[ЦЕЛЕВОЕ_ДЕЙСТВИЕ]"
2. Услуги: [КОЛИЧЕСТВО] карточек — [СПИСОК_УСЛУГ]
3. Преимущества: [4 ПУНКТА]
4. Отзывы: [3-4 КАРТОЧКИ]
5. Форма заявки: имя, телефон, [ДОППОЛЯ]
6. Контакты: [АДРЕС], [ТЕЛЕФОН], время работы [ЧАСЫ]
7. Footer: логотип, навигация, соцсети

Дизайн: [СТИЛЬ], цвета [ЦВЕТ_1] + [ЦВЕТ_2].
Tailwind CSS, shadcn/ui, адаптивный макет.

Этап 3: Правки (30-60 минут). Обычно 2-3 раунда:

Внеси изменения в лендинг:
1. [ПРАВКА_1 — например, "Замени слоган на..."]
2. [ПРАВКА_2 — например, "Добавь блок акций"]
3. [ПРАВКА_3 — например, "Измени цвет кнопок"]

Этап 4: Деплой (15 минут). В Lovable нажмите "Deploy" → "Deploy to Vercel". Отправьте клиенту ссылку.

Аналогия

Ваш процесс — как конвейер. Бриф → Lovable → правки → деплой. С каждым клиентом вы быстрее.

Ожидаемый результат: сайт опубликован за 2-4 часа. Клиент доволен.

Шаг 5: Ценообразование и допродажи

Не продавайте просто «сайт». Продавайте пакеты.

ПакетЦенаЧто входит
Лёгкий15-25К₽Лендинг 1 страница, форма заявки
Стандарт30-50К₽Лендинг + 2-3 доп. страницы + SEO-тексты
Бизнес50-80К₽Полный сайт + калькулятор + интеграции

Допродажи (ежемесячный доход): поддержка 3-5К₽/мес, обновление контента 2-3К₽/мес, SEO 5-10К₽/мес.

Про деньги

Берите предоплату 50%. Остаток — после сдачи. Так вы защищены от «передумал».

Шаг 6: Масштабирование

Первые 3-5 проектов позади. Пора расти.

  1. Портфолио на сайте — создайте сайт-портфолио в Lovable
  2. Отзывы — просите видеоотзыв после каждого проекта
  3. Специализация — станьте «тем, кто делает сайты для стоматологий»
  4. Шаблоны — сохраняйте промпты для каждой ниши, ускоряйтесь
Создай сайт-портфолио веб-разработчика на React.

Структура:
1. Hero: "[ВАШЕ_ИМЯ]", "Создаю сайты для [НИША] за 3 дня",
   кнопка "Обсудить проект"
2. Портфолио: [КОЛИЧЕСТВО] карточек (скриншот, описание, ссылка)
3. Услуги и цены: 3 пакета (таблица)
4. Отзывы клиентов: [3-5] карточек
5. Форма заявки: имя, телефон, описание задачи
6. Footer: контакты, соцсети

Дизайн: минималистичный, тёмная тема, акцентный цвет [ЦВЕТ].
Итог

Вы не программист. Вы — предприниматель с инструментом. Клиенту важен результат вовремя.

Что дальше

Фриланс запущен. Вот идеи для роста:

1. Пакетные продажи — предложите 5 клиникам в одном городе.

2. Партнёрства — маркетологи приводят клиентов, вы делаете сайт.

3. Агентство — наймите помощника для общения с клиентами.

4. Обучение — запишите курс «Как зарабатывать на сайтах без кода».

5. Подписочная модель — поддержка + обновления = ежемесячный доход.

6. Продажа шаблонов — продавайте промпты другим фрилансерам.

✏️Упражнение: Создайте свой первый демо-сайт для портфолио

Пример плана действий:

  1. Выберите нишу: салоны красоты
  2. Откройте Lovable
  3. Используйте промпт для стоматологии — замените на данные салона
  4. Опубликуйте через Deploy → Vercel
  5. Повторите для двух других ниш

Через 3 часа у вас 3 сайта. Можно искать первого клиента.

Готово! Вы запустили фриланс на создании сайтов. Три демо-проекта. Шаблон общения с клиентом. Рабочий процесс от брифа до сдачи. Тарифная сетка. Всё без единой строчки кода — только вы и Lovable.

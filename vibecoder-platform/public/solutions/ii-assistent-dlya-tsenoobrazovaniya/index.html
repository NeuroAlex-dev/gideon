ИИ-ассистент для ценообразования

«Сколько стоит ваша услуга?» — и вы замираете. Скажете дорого — клиент уйдёт. Скажете дёшево — будете работать в минус. Ставите цену наугад и каждый раз сомневаетесь.

Сейчас мы построим ассистента, который анализирует рынок, конкурентов и ценность вашего продукта — и даёт обоснованную цену с аргументами.

Что понадобится

  • Lovable — AI-конструктор для создания интерфейса ассистента
  • Google AI Studio или Claude — для ИИ-анализа рынка и расчёта цены
Аналогия

Ценообразование — это как оценка квартиры. Можно назвать цену наугад и продавать год. А можно вызвать оценщика, который посмотрит район, площадь, ремонт, аналоги — и назовёт правильную цену. ИИ-ассистент — ваш оценщик для продуктов и услуг.

Быстрый старт

Откройте Google AI Studio или Claude и вставьте промпт из Шага 3. Опишите продукт — получите анализ цены за 3 минуты.

Шаг 1: Подготовка

Соберите данные:

  1. Продукт/услуга: описание, что включено
  2. Себестоимость: ваши затраты (время, материалы, инструменты)
  3. Конкуренты: 3-5 аналогов с ценами
  4. Аудитория: B2B/B2C, платёжеспособность
  5. Цель: максимум прибыли, максимум клиентов, премиум-позиционирование

Шаг 2: Создание ассистента в Lovable

  1. Откройте Lovable
  2. Создайте новый проект
  3. Вставьте промпт:
Создай ИИ-ассистента для ценообразования на React с Tailwind CSS.

Wizard анализа:

Шаг 1 — Продукт:
— Название продукта/услуги
— Описание: что включено (textarea)
— Тип: товар, услуга, подписка, курс, консалтинг
— Ваши затраты (себестоимость): сумма или часы × ставка

Шаг 2 — Рынок:
— Конкуренты: 3 поля (название + цена)
— Сегмент: эконом / средний / премиум
— Аудитория: B2B / B2C
— Регион: Россия / СНГ / международный

Шаг 3 — Ценность:
— Какой результат получает клиент (textarea)
— Сколько клиент экономит/зарабатывает благодаря вашему продукту
— Уникальность: что есть у вас, чего нет у конкурентов

Результат:
— Рекомендуемая цена (диапазон: мин — оптимум — макс)
— Визуализация: шкала с конкурентами и вашей позицией
— Обоснование: почему именно такая цена (5 аргументов)
— Стратегии:
  - «Проникновение» — низкая цена для набора базы
  - «Ценность» — средняя цена с акцентом на результат
  - «Премиум» — высокая цена с позиционированием эксперта
— Для каждой стратегии: цена + когда использовать + плюсы/минусы

Дизайн: аналитический, серьёзный. Светлая тема. Зелёные акценты. Шрифт Inter.

Ожидаемый результат: wizard → анализ рынка → рекомендуемая цена с обоснованием.

Шаг 3: Промпт для анализа цены

Этот промпт работает в Google AI Studio или Claude:

Ты — консультант по ценообразованию с 15-летним опытом.

Проанализируй и определи оптимальную цену:

— Продукт: {название и описание}
— Себестоимость: {сумма или часы × ставка}
— Конкуренты: {название — цена} (3-5 штук)
— Сегмент: {эконом / средний / премиум}
— Аудитория: {B2B / B2C, описание}
— Результат для клиента: {что получает}
— Экономия/заработок клиента: {сумма, если есть}

Сделай анализ:

1. ПОЗИЦИОНИРОВАНИЕ НА РЫНКЕ:
— Где вы среди конкурентов (дешевле / на уровне / дороже)
— Ваши преимущества, оправдывающие цену

2. РЕКОМЕНДУЕМАЯ ЦЕНА:
— Минимальная (покрывает затраты + минимальная маржа)
— Оптимальная (баланс прибыли и спроса)
— Максимальная (при премиум-позиционировании)

3. ОБОСНОВАНИЕ (5 аргументов):
— Почему клиенту это стоит таких денег
— ROI для клиента: «Заплатите X — получите/сэкономите Y»

4. ТРИ СТРАТЕГИИ:
— Проникновение: цена, когда использовать, риски
— Ценность: цена, когда использовать, как позиционировать
— Премиум: цена, что добавить для обоснования

5. ТАРИФНАЯ СЕТКА:
— 3 тарифа: базовый, стандарт, VIP
— Для каждого: что входит, цена, для кого
— Якорный эффект: средний тариф должен быть самым привлекательным
Совет

Вставьте этот промпт в Google AI Studio или Claude — получите полный анализ ценообразования за 3 минуты. Для систематической работы — используйте интерфейс.

Шаг 4: Тарифная матрица

Добавь конструктор тарифов:

1. Генератор тарифной сетки:
— Ввод: список функций продукта
— ИИ распределяет функции по 3 тарифам
— Базовый: ядро продукта
— Стандарт: ядро + ценные дополнения
— VIP: всё + персональное + бонусы

2. Визуализация:
— Таблица сравнения тарифов (как на SaaS-сайтах)
— Рекомендуемый тариф выделен
— Экономия при годовой оплате

3. Психология цен:
— 9 990₽ вместо 10 000₽ (эффект «девяток»)
— Перечёркнутая старая цена
— «Экономия 20% при годовой оплате»
— «Цена за день: всего 33₽» (декомпозиция)

Шаг 5: Калькулятор для клиента

Добавь публичный калькулятор:

1. Калькулятор стоимости:
— Клиент выбирает параметры (чекбоксы, слайдеры)
— Цена пересчитывается в реальном времени
— «Ваша конфигурация: 47 500₽»

2. ROI-калькулятор:
— «Сколько вы тратите сейчас?»
— «С нашим продуктом будете тратить:»
— «Экономия за год: {сумма}₽»
— «Окупаемость: через {N} месяцев»

3. Виджет для сайта:
— Встраиваемый калькулятор
— Код для копирования (iframe)
Важно

Не гонитесь за низкой ценой. Дешёвый продукт = дешёвый в восприятии. Если ваш продукт реально даёт результат — ставьте справедливую цену. Клиенты, которые платят больше, ценят продукт больше.

Шаг 6: Деплой (публикация)

Проверь дизайн:
— Шкала цен с конкурентами — наглядная
— Тарифная сетка — средний тариф выделен
— Мобильная версия: тарифы читаются с телефона
— Пустое состояние: «Опишите продукт — получите оптимальную цену»

Два варианта публикации:

  1. Бесплатно — на поддомене Lovable
  2. Свой домен — подключите домен
Готово!

У вас есть ассистент по ценообразованию. Обоснованная цена вместо «пальцем в небо». Три стратегии и тарифная сетка — для уверенных продаж.

Что дальше

  • Подключите Google AI Studio / Claude API для анализа прямо в интерфейсе
  • Добавьте мониторинг конкурентов: автоматическое отслеживание изменений цен
  • Создайте A/B тест цен: две группы клиентов видят разные цены
  • Настройте динамическое ценообразование: скидки в низкий сезон, наценка в пик
  • Добавьте калькулятор скидок: «Какую скидку дать, чтобы не уйти в минус?»

✏️Упражнение: Практика: определите цену продукта

Опишите свой продукт и проведите анализ. Сравните текущую цену с рекомендацией ИИ.

Пример для фрилансер-дизайнера:

Продукт: Дизайн лендинга под ключ Себестоимость: 20 часов × 1 500₽ = 30 000₽ Конкуренты: Студия А — 120 000₽, фрилансер Б — 40 000₽, конструктор — 5 000₽

Рекомендация ИИ: — Минимальная: 45 000₽ (маржа 50%) — Оптимальная: 65 000₽ (на уровне рынка + ваш опыт) — Премиум: 95 000₽ (с аналитикой, прототипом и 2 раундами правок)

Тарифы:

ТарифЦенаЧто входит
Базовый45 000₽Дизайн + адаптив
Стандарт65 000₽+ прототип + аналитика + 2 правки
VIP95 000₽+ вёрстка + запуск + 30 дней поддержки

Вывод: текущая цена 40 000₽ — заниженная. Поднять до 65 000₽ за стандарт — обоснованно опытом и результатами.

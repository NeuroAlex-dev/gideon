Бот для кросс-продаж между партнёрами

У вас есть партнёры со смежной аудиторией. Вы договорились рекомендовать друг друга. Но на практике — рекомендуете раз в полгода, когда вспомните. Система не работает.

Сейчас мы построим бота, который автоматически рекомендует партнёрские продукты вашим клиентам в нужный момент. И наоборот — партнёр рекомендует вас своим.

Что понадобится

  • Lovable — AI-конструктор для создания системы кросс-продаж
Аналогия

Кросс-продажи между партнёрами — это как обмен визитками в бизнес-клубе, но автоматический. Клиент купил у вас дизайн сайта — бот предлагает услуги копирайтера-партнёра. Клиент купил у копирайтера текст — бот рекомендует ваш дизайн.

Шаг 1: Подготовка

Определите:

  1. Партнёры: кто работает с той же аудиторией, но не конкурирует
  2. Товары для обмена: что рекомендуете друг друга
  3. Момент рекомендации: после покупки, при определённом запросе, по времени
  4. Условия: процент от продажи, фикс, бартер
  5. Ссылки: UTM-метки для отслеживания переходов
Совет

Идеальный партнёр — тот, чей продукт естественно дополняет ваш. Дизайнер + копирайтер. Фитнес-тренер + нутрициолог. Бухгалтер + юрист. Клиенту нужны оба.

Шаг 2: Создание системы в Lovable

  1. Откройте Lovable
  2. Создайте новый проект
  3. Вставьте промпт:
Создай систему партнёрских кросс-продаж на React с Tailwind CSS.

Дашборд партнёрской программы:
— Мои партнёры: список с аватарами, названием, нишей
— Статистика: переходы, продажи, заработок от партнёров
— Мои рекомендации: что рекомендую партнёрам

Настройка партнёра:
— Информация: название, описание, логотип, ссылка
— Что рекомендуем: список товаров/услуг партнёра
— Условия: % от продажи (или фикс)
— Рекомендательный виджет: текст + баннер + ссылка
— Момент показа: после покупки / на странице «спасибо» / в email

Витрина рекомендаций (публичная):
— Страница «Партнёры»: карточки партнёров
— Каждая карточка: логотип, описание, спецпредложение
— Кнопка «Перейти» (с UTM-меткой)

Аналитика:
— Клики по каждому партнёру
— Продажи через партнёрские ссылки
— Кто из партнёров приносит больше всего

Дизайн: деловой, чистый. Светлая тема. Зелёные акценты (деньги). Шрифт Inter.

Ожидаемый результат: управление партнёрами → виджеты рекомендаций → аналитика продаж.

Шаг 3: Виджеты рекомендаций

Добавь виджеты для разных точек контакта:

1. Попап после покупки:
— «Спасибо за покупку! Вам может быть полезно:»
— Карточка партнёра: логотип + описание + спецпредложение
— «Скидка 15% по нашей рекомендации»
— Кнопка «Узнать подробнее»

2. Блок в email:
— HTML-блок для вставки в рассылку
— «Рекомендуем нашего партнёра: [название]»
— Кнопка + UTM-метка

3. Баннер на сайте:
— Горизонтальный баннер: «Нужен [услуга]? Наш партнёр [имя] сделает со скидкой»
— Sidebar-виджет: маленькая карточка
— Встраиваемый код (iframe / скрипт)

Шаг 4: Партнёрский кабинет

Добавь кабинет для партнёра:

1. Вход:
— Каждый партнёр получает ссылку-приглашение
— Свой кабинет с логотипом и данными

2. Статистика партнёра:
— Сколько переходов от вас
— Сколько продаж через ваши рекомендации
— Заработок: сумма и история

3. Взаимные рекомендации:
— Партнёр видит, что рекомендует о вас
— Может загрузить свои баннеры и тексты
— Может изменить спецпредложение

4. Выплаты:
— Баланс: сколько накопилось
— История выплат
— Кнопка «Запросить выплату»
Важно

Договоритесь об условиях до запуска. Кто кому сколько платит, в какие сроки, за что именно. Партнёрства без чётких условий разваливаются за месяц.

Шаг 5: Автоматические сценарии

Добавь автоматические рекомендации:

1. По триггеру:
— Клиент купил дизайн → рекомендовать копирайтера
— Клиент купил курс → рекомендовать наставника
— Клиент задал вопрос про [тема] → показать партнёра

2. По времени:
— Через 7 дней после покупки: «Кстати, наш партнёр...»
— Раз в месяц: дайджест партнёрских предложений

3. Умные рекомендации:
— Не показывать партнёра, если клиент уже перешёл
— Ротация: если 3 партнёра — показывать по очереди
— A/B тест: какой формат рекомендации конвертит лучше

Шаг 6: Деплой (публикация)

Проверь дизайн:
— Виджеты рекомендаций ненавязчивые (не раздражают)
— Партнёрский кабинет — понятный даже без инструкции
— Мобильная версия: карточки партнёров адаптированы
— Пустое состояние: «Добавьте первого партнёра»

Два варианта публикации:

  1. Бесплатно — на поддомене Lovable
  2. Свой домен — подключите домен
Готово!

У вас есть система кросс-продаж. Партнёры рекомендуют вас, вы — их. Дополнительный доход без расходов на рекламу.

Что дальше

  • Масштабируйте: добавьте 5-10 партнёров из смежных ниш
  • Создайте партнёрскую страницу: «Стать нашим партнёром» (заявка)
  • Настройте автоматические выплаты через платёжный шлюз
  • Добавьте реферальную программу для клиентов: «Приведи друга — оба получите бонус»
  • Запустите совместные акции: бандлы из продуктов разных партнёров

✏️Упражнение: Практика: запустите первое партнёрство

Найдите одного партнёра со смежной аудиторией. Настройте взаимные рекомендации и запустите на 2 недели.

Пример: дизайнер + копирайтер.

Вы (дизайнер): — После сдачи дизайна клиенту: «Нужны тексты для сайта? Наш партнёр Анна — копирайтер, специализируется на лендингах. Скидка 10% по нашей рекомендации» — UTM-ссылка: site.com?utm_source=designer_ivan

Партнёр (копирайтер Анна): — После сдачи текстов: «Нужен дизайн? Иван сделает лендинг под ключ. Скидка 10% от нас» — UTM-ссылка: ivan-design.com?utm_source=copywriter_anna

Результат за 2 недели: — Вы → Анне: 5 переходов, 2 продажи = 8 000₽ комиссии вам — Анна → вам: 3 перехода, 1 продажа = 12 000₽ выручки

Оба довольны. Клиенты получают полный комплекс услуг.

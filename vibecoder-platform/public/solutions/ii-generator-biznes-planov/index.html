ИИ-генератор бизнес-планов

Бизнес-план нужен всем: банку, инвестору, себе для ясности. Но писать его — мучение. Финмодель, анализ рынка, стратегия. Недели работы. Создадим генератор, который соберёт бизнес-план с финмоделью за 30 минут. Пользователь отвечает на вопросы — получает готовый документ.

Что понадобится

  • Lovable — AI-конструктор сайтов и приложений
  • Структура бизнес-плана — какие разделы включить
  • Примеры данных — для демонстрации генерации
Аналогия

Генератор бизнес-планов — это навигатор для предпринимателя. Вы вводите точку назначения — «кофейня в центре города». Навигатор строит маршрут: расходы, доходы, сроки окупаемости. Без навигатора вы едете наугад. С ним — по карте.

Шаг 1: Подготовка

Прежде чем открыть Lovable, определите структуру бизнес-плана и формулы расчёта.

Выпишите в блокнот:

  1. Разделы плана — какие блоки включить в документ
  2. Вопросы пользователю — что спрашиваем для генерации
  3. Формулы — прибыль, окупаемость, точка безубыточности

Пример структуры бизнес-плана:

РазделЧто содержитОткуда данные
Резюме проектаСуть бизнеса в 3 абзацахОтветы пользователя
Анализ рынкаОбъём, конкуренты, ЦАВвод пользователя
Продукт/услугаОписание, ценыВвод пользователя
ФинмодельРасходы, доходы, прибыльРасчёт по формулам
МаркетингКаналы, бюджетШаблон + ввод
РискиУгрозы и план действийШаблон по нише
Совет

Начните с 5–6 разделов. Главное — финмодель и описание продукта. Детализацию добавите позже.

Шаг 2: Создание проекта в Lovable

Откройте Lovable, создайте новый проект и вставьте промпт:

Создай генератор бизнес-планов на React.

Требования:
1. Пошаговая форма (wizard) с прогресс-баром:
   - Шаг 1: название бизнеса, ниша, описание, город
   - Шаг 2: продукт, средний чек, целевая аудитория
   - Шаг 3: стартовые инвестиции, ежемесячные расходы
     (аренда, зарплаты, маркетинг), кол-во клиентов
   - Шаг 4: каналы продвижения, бюджет на маркетинг
2. Генерация бизнес-плана:
   - Кнопка "Сгенерировать" → документ с разделами:
     резюме, продукт, ЦА, финмодель, маркетинг, риски
3. Финансовая модель (автоматический расчёт):
   - Таблица: расходы, доходы, прибыль по месяцам
   - Точка безубыточности, ROI
   - График прибыли на 12 месяцев (Recharts)
4. Данные в localStorage
5. Адаптивный дизайн
6. Tailwind CSS, shadcn/ui и Recharts

Замените ниши и поля на свои. Нажмите Enter.

Важно

Не закрывайте вкладку во время генерации. Если закроете — процесс прервётся и придётся начинать заново.

Ожидаемый результат: Пошаговая форма, кнопка генерации и готовый бизнес-план с финмоделью и графиками.

Шаг 3: Финансовая модель

Финмодель — сердце бизнес-плана. Сделаем расчёты подробнее.

Улучши финансовую модель:

1. Таблица расходов (редактируемая):
   - Категории: аренда, зарплаты, закупки, маркетинг, налоги
   - Для каждой: сумма в месяц, разовая / ежемесячная
   - Итого: стартовые + ежемесячные расходы
2. Таблица доходов:
   - Средний чек × кол-во клиентов = выручка
   - Маржинальность (%), чистая прибыль
   - Рост клиентов: +10% каждый месяц (настраиваемый)
3. Ключевые метрики (карточки):
   - Точка безубыточности, ROI за 12 месяцев,
     чистая прибыль за год
4. Графики (Recharts):
   - Линейный: выручка vs расходы по месяцам (12 мес.)
   - Область: накопленная прибыль (когда выходим в плюс)
5. Сценарии: оптимистичный (+20%), базовый,
   пессимистичный (−20%) с переключателем
Про точку безубыточности

Точка безубыточности — месяц, когда накопленный доход покроет все расходы. Инвесторы смотрят на неё в первую очередь. Чем раньше — тем лучше.

Ожидаемый результат: Редактируемые таблицы расходов и доходов. Три сценария. Графики прибыли. Точка безубыточности на карточке.

Шаг 4: Шаблоны по нишам

У кофейни и IT-стартапа разные расходы. Добавим готовые шаблоны.

Добавь шаблоны бизнес-планов по нишам:

1. Готовые шаблоны (предзаполняют форму):
   - "Кофейня" — аренда 150 000, оборудование 500 000,
     средний чек 350, 40 клиентов/день
   - "Интернет-магазин" — без аренды, закупка 200 000,
     средний чек 2 500, 100 заказов/месяц
   - "Онлайн-курс" — без аренды, съёмка 100 000,
     цена 5 000, 50 студентов/месяц
   - "Салон красоты" — аренда 100 000, оборудование 300 000,
     средний чек 2 000, 15 клиентов/день
2. Экран выбора: карточки с иконкой и описанием,
   при выборе — форма заполняется демо-данными
3. Кнопка "Сохранить как шаблон" для своих пресетов
Совет про шаблоны

Шаблон — отправная точка, не финальный план. Замените цифры на свои: реальная аренда, зарплаты, цены поставщиков. Точные данные — точный план.

Ожидаемый результат: Четыре шаблона по нишам. Форма заполняется автоматически. Можно сохранять свои пресеты.

Шаг 5: Экспорт и презентация

Бизнес-план должен выглядеть профессионально.

Добавь экспорт и режим презентации:

1. PDF: титульная, оглавление, все разделы с графиками
2. CSV финмодели для работы в Excel
3. Режим презентации:
   - Полноэкранные слайды: резюме → продукт →
     рынок → финмодель → итоги
4. История планов: список с датой, кнопка "Дублировать"
Про режим презентации

Режим презентации — как костюм для собеседования. Инвестор не читает таблицу на 50 строк. Он смотрит слайды: цифры крупно, графики наглядно.

Ожидаемый результат: PDF с оформлением. CSV для Excel. Режим презентации для инвесторов.

  1. Купите домен (например, bizplan.myservice.ru)
  2. В настройках Vercel добавьте домен и настройте DNS
Про точность расчётов

Генератор считает по формулам: выручка = чек x клиенты, прибыль = выручка − расходы. Для банка доработайте цифры с бухгалтером. Для старта — достаточно.

Что дальше

Базовый генератор готов. Вот как его развивать:

1. AI-генерация текстов — подключите OpenAI API. Нейросеть напишет описание продукта и анализ рынка.

2. Данные по рынку — автоподгрузка статистики из открытых источников. Средний чек по нише, объём рынка.

3. Сравнение версий — бизнес-план v1 vs v2. Что изменилось, стало ли лучше.

4. Командная работа — несколько соавторов редактируют один план. Комментарии и история.

5. Питч-дек — автоматическая генерация 10 слайдов для инвестора из бизнес-плана.

6. Трекер реальности — сравнивайте план с фактом после запуска. Где попали, где промахнулись.

Итог

Вы создали генератор бизнес-планов за 2–4 часа. Финмодель считается автоматически. Три сценария показывают диапазон. PDF готов для инвестора. Старт бизнеса — без недель подготовки.

✏️Упражнение: Создайте бизнес-план для своей идеи

Выберите шаблон, ближайший к вашей нише. Заполните реальными цифрами: аренда, зарплаты, цены конкурентов. Ответьте: через сколько месяцев выйдете в плюс? Какой ROI за год? Какая статья расходов самая большая?

  1. Точка безубыточности — карточка на экране финмодели. Больше 12 месяцев — пересмотрите расходы. Норма: 6–9 месяцев.
  2. ROI за год — если ниже 30%, подумайте, стоит ли вложение. Выше 50% — отличная идея. Проверьте на пессимистичном сценарии.
  3. Главная статья расходов — отсортируйте таблицу по сумме. Можно ли начать без офиса? Фрилансеры вместо штата?

Сделайте три версии: минимальный старт, средний, полный. Сравните точки безубыточности.

Готово! Ваш генератор бизнес-планов работает. Данные вводятся пошагово, финмодель считается автоматически, документ экспортируется в PDF. От идеи до плана — 30 минут вместо недель.

Маркетплейс услуг

У вас команда экспертов, но каждый продаёт через личные переписки? Создадим площадку, где каждый эксперт имеет профиль. Клиент выбирает специалиста и бронирует напрямую.

Что понадобится

  • Lovable — AI-конструктор сайтов и приложений
  • Список экспертов — имена, специализации, описания услуг, фотографии
Аналогия

Маркетплейс услуг — как торговый центр. Каждый эксперт — отдельный магазин с витриной. Клиент заходит к нужному мастеру. Без маркетплейса — все стоят в одном коридоре без вывесок.

Шаг 1: Подготовка

Прежде чем открыть Lovable, соберите данные о команде.

Заполните таблицу на каждого эксперта: имя, специализация, описание (2-3 предложения), услуги с ценами, фото.

Пример для студии красоты:

ЭкспертСпециализацияУслугаЦена
МарияСтилистСтрижка женская2 500 руб.
АлексейБарберСтрижка мужская1 800 руб.
ОльгаКосметологЧистка лица3 500 руб.
Совет

Подготовьте данные минимум по 3 экспертам. Площадка с одним специалистом — не маркетплейс, а визитка.

Шаг 2: Создание проекта в Lovable

Откройте Lovable, создайте проект и вставьте промпт:

Создай маркетплейс услуг на React.

Структура:
1. Главная страница:
   - Hero-баннер: название "[НАЗВАНИЕ_КОМПАНИИ]",
     подзаголовок "Выберите эксперта и запишитесь онлайн"
   - Поиск по имени эксперта или типу услуги
   - Фильтры: категория услуг, ценовой диапазон, рейтинг
   - Сетка карточек экспертов (фото, имя, специализация, рейтинг, цена от)

2. Профиль эксперта (отдельная страница):
   - Фото, имя, описание, рейтинг со звёздами
   - Список услуг с ценами и длительностью
   - Отзывы клиентов
   - Кнопка "Записаться"

3. Форма записи:
   - Выбор услуги, даты, времени
   - Имя клиента, телефон, комментарий
   - Кнопка подтверждения

4. Адаптивный дизайн для мобильных и десктопа
5. Цвета: [ОСНОВНОЙ_ЦВЕТ] и белый
6. Используй Tailwind CSS и shadcn/ui

Замените плейсхолдеры на свои данные. Нажмите Enter. Не закрывайте вкладку — генерация займёт 2–3 минуты.

Ожидаемый результат: Главная с карточками экспертов, страницы профилей, форма записи.

Шаг 3: Каталог экспертов и фильтрация

Настроим удобный поиск нужного мастера.

Улучши каталог экспертов на главной:

1. Категории услуг — кнопки-табы сверху:
   [ВСЕ], [КАТЕГОРИЯ_1], [КАТЕГОРИЯ_2], [КАТЕГОРИЯ_3]
   При нажатии — фильтрация карточек без перезагрузки

2. Сортировка: по рейтингу, по цене (от дешёвых), по популярности

3. Карточка эксперта:
   - Фото (круглое), имя, специализация
   - Рейтинг (звёзды + число отзывов)
   - Цена от [МИНИМАЛЬНАЯ_ЦЕНА]
   - Значок "Свободен сегодня" если есть слоты
   - Кнопка "Подробнее" → переход в профиль

4. На мобильных — 1 карточка в ряд, на планшетах — 2, на десктопе — 3
Значок доступности

Значок «Свободен сегодня» увеличивает конверсию. Клиент видит: можно записаться прямо сейчас. Это снимает барьер «а вдруг все занято».

Ожидаемый результат: Фильтры работают мгновенно. Карточки красиво отображаются на любом экране.

Шаг 4: Профиль эксперта и бронирование

Страница эксперта — место, где клиент принимает решение.

Улучши страницу профиля эксперта:

1. Шапка профиля:
   - Крупное фото, имя, специализация
   - Рейтинг: звёзды + "[ЧИСЛО] отзывов"
   - Стаж: "Опыт [ЧИСЛО] лет"
   - Значки: "Топ мастер", "Быстрый ответ"

2. Блок услуг:
   - Карточки: название, описание, длительность, цена
   - Кнопка "Записаться" на каждой карточке

3. Отзывы клиентов:
   - Имя, дата, рейтинг, текст
   - Средний балл сверху блока

4. Бронирование (появляется при нажатии "Записаться"):
   - Календарь с доступными датами
   - Слоты времени (только свободные)
   - Форма: имя, телефон, комментарий
   - Итого: услуга, дата, время, цена
   - Кнопка "Подтвердить запись"
Совет

Добавьте 3-5 отзывов на каждого эксперта. Даже шаблонные отзывы работают лучше, чем пустой блок.

Ожидаемый результат: Профиль вызывает доверие. Клиент читает отзывы и бронирует за минуту.

Шаг 5: Панель управления для владельца

Вам нужно управлять экспертами, видеть записи и статистику.

Добавь панель администратора:

1. Авторизация по паролю (страница /admin)

2. Дашборд: записей сегодня, за неделю, самый популярный эксперт, доход

3. Управление экспертами:
   - Таблица: имя, специализация, статус (активен/неактивен)
   - Кнопки: редактировать, скрыть из каталога, добавить нового

4. Список записей:
   - Таблица: дата, время, клиент, эксперт, услуга, статус
   - Фильтры: по эксперту, по дате, по статусу

5. Email-уведомления владельцу и эксперту при новой записи
Про панель управления

Панель админа — это пульт управления бизнесом. Вы видите, кто загружен, а кто простаивает. Без пульта — работаете вслепую.

Ожидаемый результат: Все записи в одном месте. Новые эксперты добавляются без правки кода.

Шаг 6: Деплой (публикация)

Вариант 1: Бесплатно (через Lovable)

  1. Нажмите "Deploy" в Lovable (вверху справа)
  2. Выберите "Deploy to Vercel"
  3. Подключите GitHub аккаунт
  4. Ваша ссылка: https://marketplace-abc.vercel.app

Вариант 2: Со своим доменом

  1. Купите домен (reg.ru или namecheap.com)
  2. Подключите в настройках Vercel
Важно

Проверьте маркетплейс на телефоне. 70% клиентов зайдут со смартфона.

Как использовать:

  • Добавьте ссылку в шапку соцсетей
  • Отправляйте клиентам вместо прайс-листа
  • Разместите QR-код на визитках

Что дальше

1. Онлайн-оплата — предоплата снижает неявки на 70%. Подключите ЮKassa или Тинькофф.

2. Личный кабинет эксперта — мастер сам управляет расписанием и видит свои записи.

3. Программа лояльности — каждая 5-я запись со скидкой. Клиенты возвращаются.

4. Рейтинг и отзывы — автоматический запрос отзыва после визита.

5. Push-уведомления — напоминание клиенту за 2 часа до записи.

Итог

Вы создали площадку для продажи услуг командой. Каждый эксперт имеет профиль. Клиенты выбирают и бронируют сами. Вы видите всё в панели управления.

✏️Упражнение: Запустите маркетплейс с тремя экспертами

Добавьте трёх экспертов с разными специализациями. Заполните профили и попросите знакомого пройти путь клиента.

Чек-лист запуска:

  1. 3 эксперта — профили заполнены, по 2-3 услуги, отзывы добавлены
  2. Фильтры — проверены по категориям и цене
  3. Бронирование — тестовая запись прошла успешно
  4. Мобильная версия — карточки читаются, кнопки нажимаются
  5. Уведомления — email приходит при новой записи

Всё работает — публикуйте и делитесь ссылкой.

Готово! Эксперты продают через профили. Клиенты выбирают и бронируют без переписок. Вы управляете командой из одной панели.

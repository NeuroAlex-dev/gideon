Для фрилансеров и агентств

Система учёта рабочего времени

«Сколько часов ушло на этот проект?» — «Ну... не знаю, дня три-четыре». Когда не считаешь время — не знаешь реальную себестоимость. Берёшь проект за 50 000₽, а по факту работаешь 80 часов. Получается 625₽/час. Дешевле, чем курьер.

Сейчас мы построим систему, которая точно считает время по проектам и задачам. Знаешь, сколько стоит час — можешь ставить правильные цены.

Что понадобится

  • Lovable — AI-конструктор для создания таймтрекера
Аналогия

Учёт времени — это как счётчик на такси. Без счётчика водитель называет цену наугад — иногда переплачиваете, иногда он в минусе. Со счётчиком — справедливая цена, основанная на реальном расстоянии. Ваш таймтрекер — счётчик для ваших проектов.

Шаг 1: Подготовка

Определите:

  1. Проекты: активные клиентские проекты
  2. Типы задач: дизайн, разработка, переговоры, правки, административное
  3. Ставка: ваша желаемая ставка за час
  4. Команда: сколько человек отслеживают время (только вы или команда)
  5. Биллинг: почасовая оплата, фикс, или для внутреннего учёта
Совет

Начните трекать время молча — не говорите клиентам. Через месяц увидите реальную картину: на какие проекты уходит больше всего, где вы работаете в минус.

Шаг 2: Создание таймтрекера в Lovable

  1. Откройте Lovable
  2. Создайте новый проект
  3. Вставьте промпт:
Создай систему учёта рабочего времени на React с Tailwind CSS.

Главный экран (таймер):
— Большой таймер по центру: 00:00:00
— Кнопки: ▶ Старт / ⏸ Пауза / ⏹ Стоп
— Выбор проекта (dropdown)
— Выбор задачи (dropdown или ввод)
— Теги: дизайн, разработка, созвон, правки (чипсы)
— Описание: что делаю (optional input)

Журнал времени:
— Сегодня: список записей (проект, задача, время, теги)
— Неделя: таблица по дням (строки — проекты, столбцы — дни)
— Месяц: сводка по проектам
— Ручное добавление: забыл включить таймер → добавить запись

Проекты:
— Список проектов: название, клиент, бюджет, потраченное время
— Прогресс-бар: потрачено X из Y часов (бюджет)
— Статус: активный / завершён / на паузе
— Ставка: руб/час для каждого проекта

Отчёты:
— По проектам: сколько часов на каждый
— По типам задач: дизайн — 40%, разработка — 30%, созвоны — 20%, правки — 10%
— По дням недели: в какие дни продуктивнее
— Финансовый: часы × ставка = доход

Дизайн: минималистичный, фокус на таймере. Светлая тема. Синие акценты. Шрифт Inter.

Ожидаемый результат: таймер → журнал → проекты с бюджетом → отчёты.

Шаг 3: Умный трекинг

Добавь автоматизацию:

1. Быстрый старт:
— Последние 5 задач: «Продолжить» в один клик
— Шорткаты: Ctrl+S — старт/пауза
— Виджет в системном трее (мини-таймер)

2. Напоминания:
— «Таймер не запущен уже 30 минут — забыли?»
— «Таймер работает 4 часа без паузы — всё ок?»
— Конец рабочего дня: «Не забудьте остановить таймер»

3. Автоматический трекинг:
— Определение активности: если за компьютером — время идёт
— Простой: если нет активности 10 мин — пауза
— Предложение: «Вы не работали 15 мин — удалить это время?»

Шаг 4: Финансовый учёт

Добавь финансовую аналитику:

1. Себестоимость проекта:
— Часы × ставка = реальная стоимость
— Сравнение: стоимость vs оплата от клиента
— Маржа: сколько реально заработали

2. Выставление счетов:
— Генерация акта: «За период X выполнено Y часов работ»
— Детализация: список задач с временем
— Экспорт: PDF-счёт для клиента

3. Ценообразование:
— Средняя ставка по проектам
— Рекомендация: «Ваша реальная ставка — 1 200₽/час. Чтобы зарабатывать 200 000₽/мес, нужно 167 часов»
— Прогноз дохода на следующий месяц
Важно

Не обманывайте себя. Считайте ВСЁ время: и работу, и созвоны, и правки, и переписку. Реальная картина бывает неприятной — но только честные данные помогут поднять цены.

Шаг 5: Командный режим

Добавь командные функции:

1. Участники:
— Приглашение по email
— Роли: владелец, менеджер, участник
— Каждый видит только свои проекты

2. Обзор команды:
— Кто что делает прямо сейчас (live)
— Часы по каждому участнику за неделю
— Загрузка: кто перегружен, кто недогружен

3. Отчёт для клиента:
— Сводка: кто сколько часов работал
— Группировка по задачам
— Профессиональный PDF с логотипом

Шаг 6: Деплой (публикация)

Проверь дизайн:
— Таймер запускается в один клик
— Журнал времени — понятный и быстрый
— Мобильная версия: трекинг с телефона
— Пустое состояние: «Создайте первый проект и запустите таймер»

Два варианта публикации:

  1. Бесплатно — на поддомене Lovable
  2. Свой домен — подключите домен
Готово!

У вас есть система учёта времени. Знаете реальную себестоимость каждого проекта. Можете ставить правильные цены и выставлять обоснованные счета.

Что дальше

  • Интегрируйте с таск-трекером (Notion, Trello) для автоматической привязки задач
  • Добавьте интеграцию с календарём: встречи автоматически становятся записями
  • Создайте еженедельный отчёт: автоматическая рассылка по понедельникам
  • Настройте бюджеты: алерт «До конца бюджета осталось 5 часов»
  • Добавьте Pomodoro-режим: 25 мин работы → 5 мин перерыв

✏️Упражнение: Практика: посчитайте реальную ставку

Трекайте время на всех проектах в течение 1 недели. Посчитайте: сколько часов реально работали и какова ваша реальная ставка (доход ÷ часы).

Пример для фрилансер-дизайнера:

Неделя трекинга:

Правки сайта (постоянный)8ч5 000₽625₽
Созвоны и переписка6ч0₽0₽
Итого44ч60 000₽1 364₽

Вывод: правки сайта за 625₽/час — убыточный проект. Или поднять цену, или отказаться. Созвоны «бесплатно» съедают 14% рабочего времени.

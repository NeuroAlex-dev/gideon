Генератор скриптов продаж

Один менеджер продаёт красиво — перевыполняет план. Второй — мямлит и теряет клиентов. Разница не в таланте, а в скрипте. Когда нет стандарта — каждый продаёт как умеет.

Сейчас мы построим генератор, который создаёт скрипты с ветвлениями. Клиент говорит «дорого» — менеджер видит 3 варианта ответа. Говорит «подумаю» — другие 3 варианта.

Что понадобится

  • Lovable — AI-конструктор для создания интерфейса скриптов
  • Google AI Studio или Claude — для ИИ-генерации текстов скриптов
Аналогия

Скрипт продаж — это как навигатор для водителя. Навигатор не делает из новичка профессионального гонщика. Но он не даёт заблудиться, показывает маршрут и предупреждает о поворотах. Скрипт — то же самое для менеджера.

Быстрый старт

Откройте Google AI Studio или Claude и вставьте промпт из Шага 3. Опишите свой продукт — получите готовый скрипт за 3 минуты.

Шаг 1: Подготовка

Соберите данные:

  1. Этапы продажи: приветствие → выявление потребностей → презентация → работа с возражениями → закрытие
  2. Типичные сценарии: холодный звонок, входящая заявка, повторная продажа
  3. Частые возражения: 5-10 самых частых
  4. Успешные фразы: что говорят лучшие менеджеры
  5. Продукт: описание, цены, преимущества, кейсы

Шаг 2: Создание генератора в Lovable

  1. Откройте Lovable
  2. Создайте новый проект
  3. Вставьте промпт:
Создай генератор и визуализатор скриптов продаж на React с Tailwind CSS.

Конструктор скрипта:

Шаг 1 — Параметры:
— Тип звонка: холодный, тёплый, входящая заявка, повторный
— Продукт: название, описание, цена
— Цель звонка: продажа, запись на встречу, квалификация
— Тон: деловой, дружелюбный, экспертный

Шаг 2 — Генерация:
— Кнопка «Сгенерировать скрипт»

Визуализация скрипта (интерактивная):
— Блок-схема: каждый этап — карточка
— Стрелки между этапами
— Ветвления: «Клиент говорит ДА → блок А» / «Клиент говорит НЕТ → блок Б»
— Каждый блок содержит:
  - Что говорит менеджер (крупным шрифтом)
  - Подсказка: на что обратить внимание
  - Варианты ответа клиента (кнопки)
— Клик по варианту → переход к следующему блоку

Режим тренировки:
— Скрипт как чат-бот: ИИ играет роль клиента
— Менеджер отвечает → ИИ реагирует по сценарию
— Оценка после тренировки: что хорошо, что улучшить

Библиотека:
— Сохранённые скрипты
— Шаблоны по нишам

Дизайн: чистый, фокус на тексте. Светлая тема. Синие акценты. Шрифт Inter.

Ожидаемый результат: конструктор → интерактивная блок-схема → тренажёр.

Шаг 3: Промпт для генерации скрипта

Этот промпт работает в Google AI Studio, Claude или внутри приложения:

Ты — бизнес-тренер по продажам с 15-летним опытом.

Создай скрипт продаж:

— Тип звонка: {холодный / тёплый / входящая заявка}
— Продукт: {название и описание}
— Цена: {сумма}
— Цель: {продажа / запись на встречу / квалификация}

Структура скрипта:

1. ПРИВЕТСТВИЕ
— Текст: что сказать
— Цель этапа: установить контакт

2. КВАЛИФИКАЦИЯ
— 3 вопроса для выявления потребности
— Что слушать в ответах

3. ПРЕЗЕНТАЦИЯ
— Формула: боль клиента → ваше решение → результат
— 3 ключевых аргумента

4. РАБОТА С ВОЗРАЖЕНИЯМИ
— Для каждого возражения:
  - «Дорого» → 2 варианта ответа
  - «Подумаю» → 2 варианта ответа
  - «У конкурентов дешевле» → 2 варианта ответа
  - «Нет времени» → 2 варианта ответа
  - «Не сейчас» → 2 варианта ответа

5. ЗАКРЫТИЕ
— 3 техники закрытия сделки
— Конкретные фразы

6. ПРОЩАНИЕ
— При согласии: подтверждение + следующий шаг
— При отказе: сохранение контакта + повод перезвонить

Правила:
— Живой язык, не роботизированный
— Каждая фраза — максимум 2 предложения
— Конкретика: цифры, кейсы, факты
Совет

Вставьте этот промпт в Google AI Studio или Claude с описанием вашего продукта. Готовый скрипт — за 3 минуты.

Шаг 4: Шаблоны по нишам

Добавь готовые шаблоны:

1. B2B-продажи:
— Холодный звонок ЛПР
— Выход на лицо, принимающее решение
— Продажа SaaS-сервиса

2. B2C-продажи:
— Входящая заявка с сайта
— Продажа онлайн-курса
— Продажа услуги (консалтинг, дизайн)

3. Специальные:
— Апсейл / кросс-сейл
— Возврат ушедшего клиента
— Продление подписки

Для каждого шаблона: готовый скрипт + адаптация под ваш продукт

Шаг 5: Аналитика и улучшение

Добавь аналитику скриптов:

1. Эффективность:
— Какой скрипт даёт лучшую конверсию
— На каком этапе чаще теряют клиентов
— Какие возражения встречаются чаще всего

2. Тренировки:
— Рейтинг менеджеров по тренировкам
— Слабые места каждого менеджера
— Рекомендации по улучшению

3. A/B тестирование:
— Два варианта скрипта → какой конвертит лучше
— Тест фраз: какое приветствие работает лучше
— Тест возражений: какой ответ эффективнее
Важно

Скрипт — это основа, а не кандалы. Менеджер должен адаптировать скрипт под конкретный разговор. Если читать дословно — клиент почувствует робота.

Шаг 6: Деплой (публикация)

Проверь дизайн:
— Блок-схема читается без зума
— Текст скрипта крупный (менеджер читает во время звонка)
— Мобильная версия: скрипт на телефоне рядом с CRM
— Пустое состояние: «Создайте первый скрипт за 5 минут»

Два варианта публикации:

  1. Бесплатно — на поддомене Lovable
  2. Свой домен — подключите домен

У вас есть генератор скриптов. Единый стандарт для всей команды. Менеджеры уверены, клиенты получают качественный сервис.

Что дальше

  • Подключите Google AI Studio / Claude API для генерации прямо в интерфейсе
  • Интегрируйте с IP-телефонией: скрипт открывается автоматически при звонке
  • Добавьте запись тренировок: менеджер тренируется с ботом, запись сохраняется
  • Создайте скрипты для чата: не только звонки, но и переписка
  • Настройте автообновление: ИИ анализирует результаты и предлагает улучшения

✏️Упражнение: Практика: создайте скрипт для входящей заявки

Опишите свой продукт и сгенерируйте скрипт для обработки входящей заявки с сайта. Протестируйте с коллегой.

Пример для digital-агентства:

Продукт: Создание сайтов под ключ, от 150 000₽

Скрипт (ключевые моменты):

  1. Приветствие: «Здравствуйте, [имя]! Видел вашу заявку на сайт. Расскажите, какая задача?»

  2. Квалификация: — «Сайт для чего? Продажи, имидж, лендинг?» — «Есть ли сейчас сайт?» — «Какие сроки? Когда нужен результат?»

  3. Презентация: «Мы делаем сайты, которые продают. Вот кейс: [компания] — рост заявок на 40% за месяц после запуска»

  4. Возражение «дорого»: «Понимаю. А сколько вам стоит один клиент с текущего сайта? Наш клиент [имя] окупил сайт за 2 месяца»

  5. Закрытие: «Давайте я подготовлю коммерческое с 3 вариантами — от базового до премиум. Когда удобно созвониться?»

Результат: конверсия из заявки в встречу выросла с 30% до 55%.

Система автоматических отзывов

Клиенты уходят довольными, но отзыв не оставляют. Знакомо? Создадим систему, которая сама просит отзыв после услуги. Количество отзывов вырастет в 3–5 раз без ваших усилий.

Что понадобится

  • Lovable — AI-конструктор, который создаёт веб-приложения за вас
  • Список клиентов — имена, контакты и дата услуги
  • Площадки для отзывов — Яндекс Карты, Google, 2ГИС, ваш сайт

Время на создание: 2–4 часа. Код писать не нужно.

Аналогия

Система отзывов — это вежливый администратор у выхода. Он не навязывается, а просто говорит: «Всё понравилось? Оставьте пару слов». Большинство соглашается, если попросить вовремя.

Шаг 1: Подготовка — определите путь клиента

Прежде чем строить систему, продумайте логику сбора.

Ответьте на вопросы:

  1. Когда просить отзыв? — через час? На следующий день?
  2. Где собирать? — Яндекс Карты, Google, 2ГИС?
  3. Что делать с негативом? — перенаправить в личное общение
  4. Как мотивировать? — скидка, бонус или просто «спасибо»

Схема: услуга оказана → клиент получает ссылку через 2–24 часа → ставит оценку → 4–5 звёзд уходят на площадку, 1–3 звезды пишут вам лично.

Совет

Идеальное время для запроса — 2–4 часа после услуги. Клиент ещё помнит впечатления, но уже не занят.

Шаг 2: Создание страницы отзывов в Lovable

  1. Перейдите на Lovable
  2. Нажмите "Create new project" → "Blank project"
  3. Вставьте промпт:
Создай систему сбора отзывов на React.

Страница оценки (главная):
- Заголовок: "Как вам [НАЗВАНИЕ КОМПАНИИ]?"
- Подпись: "Ваш отзыв помогает нам становиться лучше"
- 5 звёзд для оценки (кликабельные, с анимацией)
- Имя клиента берётся из URL-параметра (?name=Иван)

Логика после оценки:
- 4-5 звёзд → страница "Спасибо! Поделитесь впечатлениями":
  - Кнопка "Оставить отзыв на Яндекс Картах" → [ССЫЛКА_ЯНДЕКС]
  - Кнопка "Оставить отзыв в Google" → [ССЫЛКА_GOOGLE]
  - Кнопка "Оставить отзыв в 2ГИС" → [ССЫЛКА_2ГИС]
  - Текстовое поле: "Или напишите здесь"

- 1-3 звезды → страница "Нам жаль, что вам не понравилось":
  - Текстовое поле: "Расскажите, что пошло не так"
  - Кнопка "Отправить" (сохраняет и НЕ публикует)
  - Текст: "Мы обязательно разберёмся"

Дизайн: светлый фон, цвет [ВАШ_ЦВЕТ], звёзды жёлтые,
крупный шрифт, минималистичный. Tailwind CSS, shadcn/ui.
Адаптивный — большинство откроют с телефона.
Важно

Негативные отзывы не должны уходить на площадки. Перенаправляйте недовольных в личную форму — так вы узнаете проблему и исправите её.

Ожидаемый результат: страница с оценкой звёздами. Довольные уходят на площадки. Недовольные пишут вам лично.

Шаг 3: Уведомления и сбор данных

Каждый отзыв должен попадать к вам. Настроим уведомления.

Добавь систему уведомлений и хранения отзывов:

1. При любой оценке сохранять:
   - Имя клиента (из URL-параметра)
   - Оценка (1-5 звёзд), текст отзыва
   - Дата и время
   - Куда перешёл (Яндекс, Google, 2ГИС или никуда)

2. Уведомление владельцу:
   - Email на [ВАШ_EMAIL]:
     Тема: "Новый отзыв: [Имя] — [Оценка] звёзд"

3. При оценке 1-3 звезды:
   - Пометка "Негативный отзыв! Свяжитесь с клиентом"

4. Telegram-уведомление:
   - Через Bot API на chat_id [ВАШ_CHAT_ID]
Зачем отслеживать переходы

Данные покажут, на какую площадку клиенты уходят чаще. Если 80% идут на Яндекс — можно убрать остальные кнопки и упростить страницу.

Ожидаемый результат: каждый отзыв сохраняется. Вы получаете уведомление. Негативные отмечаются отдельно.

Шаг 4: Админ-панель и аналитика

Панель управления для отслеживания отзывов и создания ссылок.

Добавь админ-панель (/admin, защищённую паролем [ПАРОЛЬ]):

1. "Дашборд":
   - Средняя оценка за неделю и месяц
   - Количество отзывов: всего, положительных, отрицательных
   - График оценок по дням (столбчатая диаграмма)

2. "Отзывы" — список всех оценок:
   - Таблица: дата, имя, оценка, текст, площадка
   - Фильтры: по оценке, по дате
   - Цветовая маркировка: зелёный (4-5), красный (1-3)

3. "Генератор ссылок":
   - Поле ввода имени клиента
   - Кнопка "Создать ссылку" → https://ваш-сайт.com/?name=Имя
   - Кнопки "Копировать" и "Отправить в WhatsApp"

4. "Шаблоны сообщений":
   - Редактируемые шаблоны запросов отзыва
   - Плейсхолдеры: {имя}, {услуга}, {ссылка}

5. "Экспорт" — выгрузка отзывов в CSV

Данные хранить в localStorage.
Совет

Генератор ссылок — ключевая функция. Создали ссылку → скопировали → отправили клиенту в один клик.

Ожидаемый результат: админка с аналитикой, списком отзывов и генератором персональных ссылок.

Шаг 5: Дизайн и мотивация

Страница должна мотивировать оставить отзыв. Каждая деталь важна.

Улучши дизайн страницы отзывов:

1. Звёзды: крупные (60px), жёлтые, с анимацией при наведении
2. Приветствие: "Иван, спасибо за визит!" (имя из URL)
3. Фоновый градиент: светлый, приятный
4. После оценки 4-5 — анимация конфетти
5. Кнопки площадок: с логотипами, крупные, контрастные
   Подпись: "Займёт 1 минуту"
6. Блок мотивации: "Ваш отзыв помогает другим людям
   выбрать хорошего специалиста"
7. На мобильном: звёзды на всю ширину, кнопки крупные
8. Прогресс: "Шаг 1 из 2" → "Почти готово!"
Про мотивацию

Конфетти после хорошей оценки — как аплодисменты после выступления. Маленький штрих, а настроение поднимается. Довольный клиент скорее дойдёт до площадки.

Ожидаемый результат: красивая страница, которую приятно открыть. Клиент хочет оставить отзыв, а не закрыть вкладку.

Шаг 6: Деплой (публикация)

Публикуем систему в интернете.

Вариант 1: Бесплатно (через Lovable)

  1. В Lovable нажмите "Deploy" → "Deploy to Vercel"
  2. Подключите GitHub аккаунт (если нет — создайте бесплатно)
  3. Получите ссылку: https://review-abc.vercel.app

Вариант 2: Со своим доменом

  1. Купите домен (например, на reg.ru)
  2. Подключите в настройках Vercel
  3. Ваша ссылка: https://otzyv.ваш-домен.рф
Важно

Отправляйте ссылку из мессенджера, а не по SMS. Кликабельность в мессенджерах — 60%. В SMS — 15%.

Как отправлять: откройте админ-панель → «Генератор ссылок» → введите имя → скопируйте → отправьте клиенту в мессенджер.

Что дальше

Базовая система работает. Вот как сделать её мощнее:

1. Автоматическая отправка. Подключите CRM: после услуги ссылка уходит клиенту сама.

2. Фото-отзывы. Добавьте загрузку фото. Отзывы с фото привлекают в 3 раза больше внимания.

3. Виджет на сайт. Показывайте лучшие отзывы на своём сайте. Обновляются автоматически.

4. QR-код в заведении. Распечатайте QR-код на стойку. Клиент сканирует и оставляет отзыв на месте.

5. Система бонусов. «Оставьте отзыв — получите скидку 5% на следующий визит».

6. Еженедельный отчёт. Письмо по понедельникам: средняя оценка, количество, тренд.

Результат

Каждый клиент получает вежливый запрос на отзыв. Довольные уходят на Яндекс и Google. Недовольные пишут вам лично — вы решаете проблему до публичного негатива. Количество отзывов растёт в 3–5 раз.

✏️Упражнение: Настройте шаблон сообщения для вашего бизнеса

Напишите 3 варианта сообщения с запросом отзыва: для WhatsApp, Telegram и SMS.

Пример для стоматологической клиники:

WhatsApp: Елена, спасибо, что доверяете нам улыбку! Поделитесь впечатлениями? Займёт 1 минуту: [ссылка]

Telegram: Елена, спасибо за визит! Будем рады отзыву: [ссылка]

SMS: Спасибо за визит! Оставьте отзыв: [ссылка]

Советы: обращайтесь по имени (отклик +20%), пишите «займёт 1 минуту», отправляйте один раз.

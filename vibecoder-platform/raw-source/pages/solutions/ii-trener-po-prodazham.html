ИИ-тренер по продажам

Менеджеры боятся звонить, теряются на возражениях и забывают скрипты. Тренинги дорогие, а практика на реальных клиентах — рискованная.

Сейчас мы построим тренажёр, где менеджер тренируется с ИИ-клиентом. Безопасно, сколько угодно раз и с обратной связью после каждой тренировки.

Что понадобится

  • Lovable — AI-конструктор для создания интерфейса тренажёра
  • Google AI Studio или Claude — для ИИ-симуляции клиента и обратной связи
Аналогия

ИИ-тренер — это как авиасимулятор для пилотов. Пилот отрабатывает посадку в шторм — без реального самолёта и пассажиров. Ошибся? Не страшно — перезапустил. Наш тренажёр — то же самое, но для продавцов. Тренируй переговоры без риска потерять клиента.

Быстрый старт без разработки

Если нужен тренажёр прямо сейчас — откройте Google AI Studio или Claude и вставьте промпт из Шага 3. ИИ сыграет роль клиента, а вы потренируетесь в чате. Интерфейс в Lovable — для масштабирования на команду.

Шаг 1: Подготовка

Определите:

  1. Сценарии тренировок:

    • Холодный звонок
    • Выявление потребностей
    • Презентация продукта
    • Работа с возражениями
    • Закрытие сделки

  2. Персонажи клиентов:

    • Лояльный (легко соглашается)
    • Скептик (всё подвергает сомнению)
    • Занятой (спешит, не хочет тратить время)
    • Агрессивный (жёсткие переговоры)

  3. Критерии оценки: что оцениваем — вопросы, аргументы, эмпатия, техника закрытия

Шаг 2: Создание тренажёра в Lovable

  1. Откройте Lovable
  2. Создайте новый проект
  3. Вставьте промпт:
Создай ИИ-тренажёр по продажам на React с Tailwind CSS.

Главная страница — выбор тренировки:
— Карточки сценариев: холодный звонок, выявление потребностей, возражения, закрытие
— Каждая карточка: иконка, название, сложность (1-3 звезды), длительность
— Выбор персонажа клиента: лояльный, скептик, занятой, агрессивный

Страница тренировки — чат-интерфейс:
— Слева: аватар и описание клиента (имя, компания, настроение)
— Центр: чат — сообщения менеджера и клиента (пузыри как в мессенджере)
— Поле ввода сообщения + кнопка отправки
— Таймер тренировки
— Кнопка «Завершить тренировку» → переход к оценке

Страница оценки:
— Общий балл: 0-100
— Разбор по критериям: вопросы (0-10), аргументы (0-10), эмпатия (0-10), закрытие (0-10)
— «Что было хорошо» — 2-3 пункта
— «Что улучшить» — 2-3 пункта с конкретными рекомендациями
— Кнопка «Попробовать снова» / «Следующий сценарий»

Дизайн: современный, как Telegram. Тёмная тема. Шрифт Inter.

Ожидаемый результат: тренажёр с выбором сценария, чатом и оценкой.

Шаг 3: Промпт для ИИ-клиента

Вот промпт, который работает в Google AI Studio, Claude или внутри приложения:

Ты — ИИ-симулятор клиента для тренировки менеджеров по продажам.

Роль: {тип_клиента} (скептик / лояльный / занятой / агрессивный)
Сценарий: {сценарий} (холодный звонок / выявление потребностей / возражения / закрытие)
Компания: среднего размера, B2B
Бюджет: ограниченный, но есть

Правила поведения:
— Веди себя как реальный клиент, не подыгрывай
— Задавай встречные вопросы
— Если менеджер слабо аргументирует — не соглашайся
— Если менеджер хорош — постепенно «оттаивай»
— Используй типичные возражения: «дорого», «подумаю», «у конкурентов дешевле»
— Длина ответов: 1-3 предложения (как в реальном разговоре)

После завершения (когда менеджер скажет «завершить»):
Оцени по шкале 0-10: вопросы, аргументы, эмпатия, техника закрытия.
Дай 2 конкретных совета по улучшению.
Подсказка

Этот промпт можно вставить прямо в Google AI Studio или Claude — и тренироваться без интерфейса. Для команды из 5+ менеджеров — стройте интерфейс в Lovable.

Шаг 4: Прогресс и статистика

Добавь систему прогресса:

1. Профиль менеджера:
— Имя, аватар, уровень (новичок → мастер)
— XP за каждую тренировку (10-50 XP в зависимости от оценки)
— Шкала прогресса до следующего уровня

2. История тренировок:
— Список: дата, сценарий, персонаж, оценка
— График: средний балл по неделям (тренд роста)
— Лучший результат по каждому сценарию

3. Слабые места:
— Радар-диаграмма: вопросы, аргументы, эмпатия, закрытие
— Подсветка: «Ваше слабое место — работа с возражениями. Рекомендуем тренировку»

4. Достижения:
— «Первая тренировка», «10 тренировок», «Балл 90+», «Прошёл все сценарии»

Шаг 5: Библиотека скриптов

Добавь библиотеку продающих скриптов:

1. Скрипты по этапам:
— Приветствие и выход на ЛПР (3 варианта)
— Выявление потребностей (5 ключевых вопросов)
— Презентация через выгоды (шаблон «Характеристика → Выгода → Доказательство»)
— Работа с возражениями (10 самых частых + ответы)
— Закрытие сделки (3 техники)

2. Подсказки во время тренировки:
— Кнопка «Подсказка» — ИИ предлагает что сказать дальше
— Используется максимум 3 раза за тренировку (иначе не учишься)

3. Режим «Разбор полётов»:
— Открыть завершённый чат
— ИИ комментирует каждое сообщение: «Тут хорошо спросили» / «Тут потеряли клиента»
Помните

Тренажёр не заменяет реальные продажи. Но 10 тренировок с ИИ-клиентом дают уверенность, которой не хватает на реальных звонках.

Шаг 6: Деплой (публикация)

Проверь весь дизайн:
— Чат выглядит как настоящий мессенджер
— Мобильная версия: тренировка с телефона (менеджеры часто в дороге)
— Быстрый ответ ИИ (< 2 секунды)
— Пустое состояние: «Выберите сценарий и начните тренировку»

Два варианта публикации:

  1. Бесплатно — на поддомене Lovable
  2. Свой домен — подключите домен (например, training.вашкомпания.ru)
Готово!

У вас есть ИИ-тренер по продажам. Менеджеры тренируются без страха, получают обратную связь и растут. Конверсия продаж растёт вместе с навыками.

Что дальше

  • Подключите Google AI Studio / Claude API для реалистичных диалогов
  • Добавьте голосовой режим: тренировка через микрофон (как звонок)
  • Настройте командный рейтинг: кто из менеджеров тренируется больше
  • Создайте свои сценарии под ваш продукт и возражения
  • Добавьте запись тренировок для разбора с руководителем
  • Интегрируйте с CRM: после проигранной сделки — рекомендация тренировки

✏️Упражнение: Практика: пройдите первую тренировку

Выберите сценарий «Холодный звонок» с клиентом-скептиком. Пройдите диалог до конца и посмотрите свою оценку.

Пример диалога (фрагмент):

Менеджер: Добрый день, Алексей! Звоню из компании X. Мы помогаем компаниям вашего размера увеличить продажи на 25%. Уделите 2 минуты?

Клиент-скептик: 25% — это громкое обещание. Откуда такие цифры?

Менеджер: Справедливый вопрос. Вот свежий кейс: компания из вашей ниши за 3 месяца... [конкретные цифры]

Оценка ИИ: 72/100

  • Вопросы: 6/10 (мало вопросов о потребностях клиента)
  • Аргументы: 8/10 (хороший кейс)
  • Эмпатия: 7/10 (признал справедливость вопроса)
  • Закрытие: 5/10 (не предложил следующий шаг)

Совет: Задайте 2-3 вопроса о бизнесе клиента, прежде чем предлагать решение.

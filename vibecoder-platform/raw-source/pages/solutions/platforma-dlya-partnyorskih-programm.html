Платформа для партнёрских программ

Для инфобизнеса

Проблема
Ручной подсчёт партнёрских комиссий — хаос и ошибки
Результат
Партнёры приводят клиентов, система считает и выплачивает комиссии автоматически
Lovable2-4 часа

Что понадобится

  • Lovable — AI-конструктор сайтов и приложений
Аналогия

Партнёрская программа — как армия торговых представителей. Только им не надо платить зарплату. Они получают процент с каждой продажи. Нет продаж — нет расходов. Есть продажи — все в плюсе. Таблица Excel тут не справится — нужна система.

Шаг 1: Подготовка

Перед сборкой платформы определите правила партнёрки.

Ответьте на пять вопросов:

  1. Какой процент комиссии получает партнёр с каждой продажи?
  2. Когда начисляется комиссия — сразу или после оплаты?
  3. Есть ли минимальная сумма для вывода?
  4. Сколько уровней партнёрской программы поддерживать?
  5. Какой срок жизни реферальной cookie (30, 60, 90 дней)?

Пример структуры комиссий:

ПродуктЦенаКомиссия партнёраСумма выплаты
Мини-курс3 000 ₽20%600 ₽
Основной курс15 000 ₽15%2 250 ₽
Наставничество50 000 ₽10%5 000 ₽
Совет

Ставьте комиссию от 15% и выше. Партнёры сравнивают программы. Если у конкурента 30%, а у вас 5% — к вам никто не придёт.

Шаг 2: Создание проекта в Lovable

  1. Откройте Lovable
  2. Нажмите «New Project»
  3. Вставьте промпт:
Создай веб-приложение «Партнёрская платформа» на русском языке.

Лендинг для партнёров:
- Заголовок: «Зарабатывайте с нами — партнёрская программа»
- Блок «Как это работает» — 3 шага с иконками
- Блок с размерами комиссий по продуктам (таблица)
- Отзывы партнёров (3 карточки)
- Кнопка «Стать партнёром» — ведёт на регистрацию

Страница регистрации партнёра:
- Форма: имя, email, телефон, ссылка на соцсети или канал
- Согласие с условиями партнёрской программы
- После регистрации — личный кабинет

Личный кабинет партнёра:
- Уникальная реферальная ссылка с кнопкой копирования
- Дашборд: переходы, регистрации, продажи, комиссия
- Баланс: начислено, ожидает, выплачено
- История начислений (таблица с фильтрами)

Дизайн: современный, минималистичный, зелёно-белая палитра.
Адаптивная вёрстка для мобильных устройств.
Совет

Замените цвета на фирменные цвета вашего бренда. Партнёры должны видеть, что платформа — часть серьёзного проекта.

Ожидаемый результат: Лендинг партнёрской программы, форма регистрации и личный кабинет с дашбордом и реферальной ссылкой.

Шаг 3: Система отслеживания продаж

Партнёр привёл клиента — система должна это зафиксировать. Без отслеживания партнёрка не работает.

Отправьте в чат Lovable:

Добавь систему отслеживания партнёрских продаж:

- При регистрации партнёра — генерация уникального кода (8 символов)
- Формат ссылки: сайт.ru/p/КОД
- При переходе по ссылке — сохранение cookie на 90 дней
- Отслеживание воронки: переход → регистрация → покупка
- Автоматическое начисление комиссии при подтверждённой оплате
- Статусы сделок: «ожидает», «подтверждена», «выплачена», «отменена»
- Кнопки «Поделиться ссылкой» в Telegram и WhatsApp

Ожидаемый результат: Каждый переход, регистрация и покупка привязаны к партнёру. Комиссии начисляются автоматически.

Зачем cookie на 90 дней

Клиент может перейти по ссылке сегодня, а купить через месяц. Cookie запоминает партнёра. Без этого — партнёр теряет честно заработанную комиссию.

Шаг 4: Выплаты и финансы

Партнёры хотят видеть деньги. Прозрачность выплат — ключ к доверию.

Напишите в чат Lovable:

Добавь систему выплат партнёрам:

- Минимальная сумма для вывода: 3000 рублей
- Кнопка «Запросить выплату» в личном кабинете
- Партнёр указывает реквизиты (карта или расчётный счёт)
- Статусы выплат: «запрошена», «в обработке», «выплачена»
- Уведомление партнёру при смене статуса (зелёный тост)
- История выплат: дата, сумма, статус, способ
- Автоматический расчёт суммы к выплате за период

Ожидаемый результат: Партнёр видит баланс, запрашивает выплату и отслеживает статус. Все расчёты прозрачны.

Важно

Добавьте период «холда» — 14 дней между начислением и возможностью вывода. Это защитит от возвратов. Клиент вернул деньги — комиссия отменяется автоматически.

Шаг 5: Админ-панель владельца

Вам нужна полная картина: кто из партнёров приносит деньги, а кто просто занимает место.

Добавь панель администратора по маршруту /admin:

- Общая статистика: партнёры, продажи, комиссии, конверсия
- График продаж через партнёров по неделям
- Таблица партнёров: имя, переходы, продажи, сумма комиссий
- Сортировка по любому столбцу
- Фильтры: по дате, по статусу, по продукту
- Кнопка одобрения/отклонения заявок на выплату
- Возможность заблокировать партнёра
- Экспорт всех данных в CSV для бухгалтерии

Ожидаемый результат: Вы видите эффективность каждого партнёра. Одобряете выплаты в один клик.

Совет

Раз в неделю проверяйте топ-5 партнёров. Предложите лучшим повышенную ставку. Это мотивирует их продвигать вас ещё активнее.

Шаг 6: Деплой (публикация)

Платформа готова. Публикуем и приглашаем партнёров.

Вариант 1 — бесплатный:

В Lovable нажмите «Share» → «Publish». Получите ссылку вида проект.lovable.app. Подходит для тестирования с первыми партнёрами.

Вариант 2 — свой домен:

  1. Купите домен (например, partners.вашсайт.ru)
  2. В настройках проекта добавьте свой домен
  3. Настройте DNS-записи у регистратора домена
Совет

Используйте поддомен. Адрес partners.вашсайт.ru выглядит солиднее, чем случайная ссылка. Партнёры доверяют знакомому бренду.

Готово

Партнёрская платформа работает. Партнёры регистрируются, получают ссылки, приводят клиентов. Система считает комиссии автоматически. Никаких таблиц, никаких ошибок в расчётах.

Что дальше

Платформа запущена. Вот направления для масштабирования:

  • Добавьте промо-материалы для партнёров — баннеры, тексты, шаблоны постов
  • Создайте многоуровневую программу — партнёр получает процент с продаж своих партнёров
  • Подключите автоматические email-отчёты партнёрам раз в неделю
  • Добавьте рейтинг партнёров с призами для лидеров
  • Интегрируйте с CRM для автоматического учёта сделок
  • Настройте вебхуки для мгновенного начисления комиссий при оплате

✏️Упражнение: Практика: запустите партнёрскую программу

Создайте партнёрскую платформу для вымышленной онлайн-школы «ПроСтарт».

Условия:

  • Курс «Основы маркетинга» стоит 12 000 рублей
  • Комиссия партнёра — 20% (2 400 рублей с продажи)
  • Минимальная сумма для вывода — 5 000 рублей
  • Cookie живёт 60 дней

Задача: соберите рабочий прототип в Lovable и проверьте полный цикл.

Проверьте:

  • Регистрация партнёра создаёт уникальную ссылку
  • Переход по ссылке фиксируется в дашборде
  • Комиссия начисляется при «покупке»
  • Админ-панель показывает корректные данные
  • Запрос на выплату создаётся и меняет статус

Используйте промпты из шагов 2-5 выше. Замените суммы и проценты на указанные в условии. Проверьте полный цикл: регистрация партнёра → получение ссылки → переход по ссылке → регистрация клиента → покупка → начисление комиссии → запрос выплаты. Откройте сайт в двух разных браузерах — один для партнёра, другой для клиента. Убедитесь, что админ-панель отображает все транзакции корректно.

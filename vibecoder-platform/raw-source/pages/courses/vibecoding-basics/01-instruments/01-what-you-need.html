Что нужно для начала

Давайте сразу к делу. Прежде чем создавать сайты с помощью ИИ, нужен набор инструментов. Не пугайтесь — большинство из них бесплатные и ставятся за пару кликов.

Аналогия

Аналогия: собираем портфель в первый класс. Ручки, тетрадки, учебники — без них в школу не пойдёшь. У нас то же самое, только вместо ручек — программы на компьютере.

Что вас ждёт в этом уроке

  • Минимальный набор — 5 инструментов, чтобы начать
  • Расширенный набор — для комфортной работы
  • Сколько стоит — от 0 до $40/мес
  • Домашнее задание — определитесь со своим набором

Блок 1: Минимальный набор (7 мин)

Вот 5 инструментов, без которых не обойтись. Это ваш базовый набор вайбкодера:

#ИнструментЗачем нуженБесплатно?
1Google ChromeПросматривать и тестировать сайты✅ Да
2VS Code или CursorРедактировать код✅ Да / Есть бесплатный план
3Claude или ChatGPTГенерировать код по вашему описанию✅ Есть бесплатные планы
4Node.jsЗапускать проекты на компьютере✅ Да
5GitСохранять версии проекта✅ Да
Аналогия

Аналогия: кухня и повар. Chrome — это ваша тарелка, куда подаётся блюдо. VS Code — кухня, где готовят. Claude — повар, который готовит по вашему заказу. Node.js — плита. Git — холодильник, чтобы ничего не пропало.

Совет

Если вы на Mac — Chrome, Node.js и Git могут быть уже установлены. Проверьте прежде чем качать!

Готово

Что поняли: для старта нужно всего 5 бесплатных программ. Не надо ничего покупать, чтобы начать.

Блок 2: Расширенный набор (5 мин)

Когда освоитесь с базовым набором, вот что добавит комфорта:

ИнструментЗачемЦена
Cursor ProРедактор кода со встроенным ИИ — не нужно переключаться~$20/мес
Claude ProБольше запросов к ИИ, быстрее ответы, умнее модели~$20/мес

Аналогия

Аналогия: автомобиль. Базовый набор — это механическая коробка. Ездить можно. Расширенный — автомат с климат-контролем. Едете туда же, но комфортнее.

Готово

Что поняли: расширенный набор не обязателен для старта. Но когда начнёте делать проекты каждый день — он сэкономит время и нервы.

Блок 3: Сколько закладывать денег? (5 мин)

Давайте честно — вайбкодинг можно начать бесплатно. Но подписки реально ускоряют работу. Вот три сценария:

🆓 Бесплатный старт (0 руб.)

  • VS Code + бесплатный Claude/ChatGPT
  • Ограничение: мало запросов к ИИ в день (примерно 10-15)
  • Для кого: попробовать и понять, нравится ли

💰 Комфортное обучение (~$20/мес)

  • Cursor Pro ИЛИ Claude Pro
  • Достаточно запросов для ежедневной практики
  • Для кого: серьёзно учиться и делать проекты

🚀 Максимальный комфорт (~$40/мес)

  • Cursor Pro + Claude Pro
  • Без ограничений — хоть 10 проектов в день
  • Для кого: уже понял, что это моё
Внимание

Не тратьте деньги сразу! Начните с бесплатного плана. Если через неделю поймёте, что запросов не хватает — тогда подключайте подписку.

Готово

Что поняли: старт — бесплатный. Подписка на ~$20/мес понадобится, когда будете практиковаться каждый день. Не раньше.

Блок 4: Что выбрать? VS Code или Cursor? (3 мин)

Это самый частый вопрос новичков. Ответ простой:

VS CodeCursor
ЦенаБесплатно$0-20/мес
ИИ внутри❌ Нужно ставить расширения✅ Встроен из коробки
Удобство для новичкаНужно настроитьРаботает сразу
ОсноваVS CodeТот же VS Code, но с ИИ
Рекомендация

Если вы полный новичок — начните с Cursor. Он сделан для вайбкодинга. ИИ уже встроен, ничего не надо настраивать.

Готово

Что поняли: Cursor = VS Code + встроенный ИИ. Для вайбкодинга это удобнее. Но если не хотите тратить — VS Code тоже подойдёт.

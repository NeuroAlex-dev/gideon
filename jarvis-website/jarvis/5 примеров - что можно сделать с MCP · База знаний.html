5 примеров - что можно сделать с MCP

7 мин · обновлено 2026-05-10
MCP превращает Агента из собеседника в рабочую лошадку. Подключили календарь - Агент планирует встречи. Подключили почту - Агент пишет и отвечает. Подключили таблицы - Агент считает.

MCP позволяет Агенту работать с внешними сервисами напрямую. Не через копирование данных туда-сюда, а через прямое подключение. Ниже пять примеров для повседневных и бизнес-задач: что подключить, как настроить, какие команды давать Агенту.

Главное сообщение

Подключил - используй

Каждый MCP-сервер дает Агенту набор новых действий. После подключения Вы просто описываете задачу обычными словами - Агент сам вызывает нужное действие из подключенного сервиса.

1. Google Calendar - планирование дня

Подключив календарь, Агент видит Ваше расписание, создает события, ищет свободные окна и напоминает о встречах.

Как подключить:

Скажите Агенту:

«Подключи MCP-сервер Google Calendar»

Агент добавит в ~/.claude.json:

{
  "google-calendar": {
    "type": "http",
    "url": "адрес_вашего_mcp_сервера",
    "headers": {
      "Authorization": "Bearer ваш_токен"
    }
  }
}

Что можно делать:

«Какие у меня встречи на сегодня?»

«Поставь созвон с Настей на четверг в 15:00, тема - обсуждение контент-плана»

«Найди свободное окно на следующей неделе для двухчасовой сессии»

«Напоминай мне каждое утро в 8:00, что запланировано на день»

2. Gmail - почта без открытия почты

Агент читает входящие, пишет черновики, отвечает на письма и сортирует по важности. Вы не открываете почтовый клиент - всё через Telegram.

Что можно делать:

«Покажи непрочитанные письма за сегодня»

«Напиши ответ Алексею: спасибо, всё получил, встречаемся в среду»

«Найди все письма от Сбербанка за последний месяц»

«Отсортируй входящие: что срочно, что может подождать, что спам»

3. Google Sheets - таблицы и отчеты

Агент работает с таблицами напрямую: читает данные, добавляет строки, считает итоги, строит сводки. Идеально для учета, CRM, финансов.

Что можно делать:

«Открой таблицу "Продажи апрель" и покажи итог по столбцу "Сумма"»

«Добавь в таблицу клиентов: Иванов Петр, +7-999-123-45-67, заявка на консультацию»

«Сделай сводку по продажам за неделю из таблицы "CRM"»

«Выгрузи список клиентов, которые не покупали больше 30 дней»

4. Pencil - дизайн макетов из Telegram

Этот MCP-сервер уже установлен на Вашем сервере из коробки. Агент умеет открывать файлы макетов (.pen), менять текст, цвета, экспортировать картинки.

Конфигурация (уже есть в ~/.claude.json):

{
  "pencil": {
    "type": "stdio",
    "command": "/home/agent/.pencil/mcp/visual_studio_code/out/mcp-server-linux-x64",
    "args": ["--app", "visual_studio_code"],
    "env": {}
  }
}

Что можно делать:

«Открой макет визитки и поменяй номер телефона на +7-999-000-00-00»

«Сделай скриншот главной страницы из макета landing.pen»

«Экспортируй логотип из макета в PNG»

5. Данные с сайтов - аналитика и мониторинг

Supadata - MCP-сервер для получения информации с веб-страниц. Полезен для мониторинга конкурентов, сбора контента, анализа рынка.

Конфигурация:

{
  "supadata": {
    "type": "http",
    "url": "https://mcp.supadata.dev/v1"
  }
}

Как подключить:

«Подключи MCP-сервер supadata, тип http, URL: https://mcp.supadata.dev/v1»

Что можно делать:

«Достань текст статьи с этого URL и сделай краткую выжимку»

«Получи расшифровку этого YouTube-видео»

«Собери описания товаров с этой страницы и оформи в таблицу»

Как подключить любой новый MCP-сервер

Для всех примеров выше формат одинаковый. Скажите Агенту:

«Подключи MCP-сервер [название], URL: [адрес]»

Или если сервер запускается локально:

«Подключи MCP-сервер [название], команда: npx [имя-пакета]»

Агент сам отредактирует ~/.claude.json и подключится к сервису. Подробная инструкция с форматами - в статье «Как подключить MCP-сервер».

Мой Вам совет

Начните с того, чем пользуетесь каждый день. Если весь день в почте - подключите Gmail. Если ведете клиентов в таблице - подключите Google Sheets. Если много встреч - Google Calendar.

Pencil уже установлен из коробки. Supadata подключается без ключа - бесплатно.

Не подключайте всё сразу. Один сервис, попробовали, привыкли - потом следующий.

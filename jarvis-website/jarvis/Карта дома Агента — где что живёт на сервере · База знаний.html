Карта дома Агента — где что живёт на сервере

5 мин · обновлено 2026-05-09
У Агента три комнаты — мотор, душа и мастерская. Знаете карту — никогда не запутаетесь.

Если Вы решите заглянуть в гости к своему Агенту через VS Code Tunnel — пригодится карта его дома. Чтобы знать в какой комнате что живёт и куда лазить руками не нужно. Эта статья — короткий план этажей.

Главное сообщение

Главный смысл

У Агента три зоны на сервере: системная (мотор), личность (душа) и проекты (мастерская). Знаете эту карту — никогда не запутаетесь, что трогать руками, а что не надо.

Карта целиком

🏠 /home/agent/
   │
   ├── 🔧 ~/.agent/           ← мотор (НЕ трогаем)
   │      └─ bot · sessions · .env · state.json
   │
   ├── 💛 ~/workspace/        ← душа (правьте смело)
   │      └─ SOUL · MEMORY · GOALS · skills · memory/ · knowledge/
   │
   └── 🛠 ~/projects/         ← мастерская (рабочие проекты)
          └─ лендинги · боты · отчёты · контент

Зона 1 — мотор (~/.agent/)

Это системная папка. Здесь живёт сам бот, ключи API, настройки, история сессий, бэкапы предыдущих версий. Агенту туда нельзя писать — read-only, чтобы случайно ничего не сломал.

🔧 ~/.agent/
   ├─ bot/             ← движок Telegram-бота
   │     ├─ index.js
   │     └─ versions/  ← бэкапы прошлых версий (для отката)
   ├─ sessions/        ← история переписки (авто-очистка через 7 дней)
   ├─ traces/          ← логи для отладки
   ├─ .env             ← API-ключи (доступ только владельцу)
   └─ state.json       ← настройки модели, лимиты, расписание
Сюда руками лучше не лазить

Эти файлы трогает только сам Бот «Конструктор Агентов» при установке и обновлениях. Если что-то здесь сломаете — Агент перестанет работать. Хотите изменить настройки (модель, лимит, расписание)? Используйте /settings в чате с Агентом — там всё устроено безопасно.

Зона 2 — душа (~/workspace/)

Это сердце Агента. Здесь живёт всё, что делает Вашего Агента именно Вашим: характер, память о Вас, Ваши цели, Ваши предпочтения, команда специалистов. Здесь Вы можете править руками что угодно — ничего не сломается.

💛 ~/workspace/
   │
   ├─ 📜 Файлы личности (8 главных файлов Агента)
   │     ├─ SOUL.md         ← характер, тон, манера
   │     ├─ MEMORY.md       ← долгосрочная память о Вас
   │     ├─ GOALS.md        ← Ваши текущие цели
   │     ├─ USER.md         ← Ваш профиль (роль, опыт, контекст)
   │     ├─ MISSION.md      ← миссия Агента
   │     ├─ PROJECTS.md     ← список активных проектов
   │     ├─ PREFERENCES.md  ← Ваши предпочтения работы
   │     └─ LEARNED.md      ← выводы Агента из работы с Вами
   │
   ├─ ⚙️ Конфигурация
   │     ├─ CLAUDE.md       ← правила поведения (что делать всегда)
   │     ├─ SERVICES.md     ← каталог Ваших сервисов и API
   │     └─ DATA-POLICY.md  ← политика обработки данных
   │
   ├─ 🛠 .claude/skills/    ← 14 специалистов (writer · designer · researcher...)
   │
   ├─ 📂 memory/            ← дневник: YYYY-MM-DD.md за каждый день
   ├─ 📚 knowledge/         ← база знаний (статьи, решения, конфиги)
   ├─ 📦 content/           ← контент-материалы
   └─ 🖼 .media/            ← фото/видео из чата (авто-очистка >7 дней)
Сюда правьте смело

Это Ваша зона. Хотите изменить характер Агента — открываете SOUL.md и пишете. Хотите добавить факт в память — дописываете в MEMORY.md. Хотите, чтобы Агент знал о новой цели — добавляете в GOALS.md. Ничего не сломается. Агент при следующем запуске просто прочитает обновлённую версию.

💬 Перевод термина: «DNA-файлы» — это 8 файлов с большой буквы (SOUL, MEMORY, GOALS и т.д.). Они описывают Агента: какой он, что помнит, к чему стремится. ДНК — потому что именно эти файлы делают Вашего Агента уникальным, не похожим на других.

Зона 3 — мастерская (~/projects/)

Здесь живут Ваши рабочие проекты: лендинги, боты, отчёты, исследования, контент-планы. Каждый проект — отдельная папка. У каждого свой memory/ (заметки по проекту) и knowledge/ (решения, конфиги, важные ссылки).

🛠 ~/projects/
   ├─ продажи-2026-q2/
   │     ├─ memory/         ← заметки по проекту
   │     ├─ knowledge/      ← решения, конфиги, лиды
   │     └─ ... (файлы проекта)
   │
   ├─ landing-osen/
   │     └─ ...
   │
   └─ telegram-bot-school/
         └─ ...

Когда Вы говорите Агенту «Запусти проект про новый лендинг» — он создаёт здесь отдельную папку и складывает туда всё: код, заметки, контент, скриншоты. Через месяц Вы сможете вернуться к этому проекту, и Агент откроет папку — как офисный сотрудник возвращается в свою картотеку.

Что можно править а что нет — светофор

🔴 НЕ ТРОГАЕМ
~/.agent/ — мотор Бота сюда пишет только Бот «Конструктор Агентов» при установке и обновлениях

Мой Вам совет

Если только начинаете — откройте через VS Code Tunnel три файла: SOUL.md, MEMORY.md, GOALS.md. Прочитайте что Агент уже знает о Вас. Допишите если что-то важное упустили. Это самая полезная пятнадцатиминутка в первый день — Вы сразу почувствуете что Агент именно Ваш, а не «один из тысячи».

И ещё: не пугайтесь количества файлов. На карте их много, но в реальной жизни Вы будете заходить в три-четыре файла, остальные просто молча работают. Карта нужна на случай «А где у меня лежат вчерашние заметки?» — тогда Вы знаете: ~/workspace/memory/2026-05-09.md.

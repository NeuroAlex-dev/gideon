Меню /settings — карта по вкладкам

7 мин · обновлено 2026-05-10
Шесть переключателей в одном окне. Знать каждый — полезно. Трогать без причины — нет.

Команда /settings в чате с Агентом открывает главное меню настроек. Это шесть вкладок, в которых живёт всё, что Вы можете крутить: какая модель Claude работает, сколько Агент может потратить за день, где у Вас часовой пояс, и куда вставлять ключи внешних сервисов. По умолчанию там уже подобран разумный набор — но знать что в каждой вкладке полезно, чтобы не лазить наугад.

Главное сообщение

Главный смысл

Шесть переключателей. По умолчанию настроены разумно — можно работать без правок месяцами.

В первый раз стоит зайти в три вкладки: 🤖 Модель (если у Вас Max — поставить Opus), 🕐 Часовой пояс (выбрать свой) и 🔑 Переменные окружения (подключить Deepgram для голоса).

Остальные вкладки — для особых случаев. Лимит вообще не трогают, если работаете через подписку.

Как открыть меню

В чате с Вашим Агентом отправьте /settings. Откроется текст «⚙️ Настройки» с inline-кнопками — это и есть меню. Нажимаете на любую кнопку — она ведёт во вложенное меню той вкладки. Кнопка «← Назад» возвращает наверх, «✖ Закрыть» — удаляет сообщение целиком.

Вкладка 1 — 🤖 Модель

Внутри Claude есть три модели:

  • 🟢 Haiku — самая лёгкая. Быстрая, дешёвая, но менее «умная». Подходит для простых задач: переформулировать текст, разбить на пункты, короткий ответ.
  • 🔵 Sonnet — золотая середина. Универсальный баланс скорости, цены и качества. После установки Агента стоит именно эта модель — безопасный выбор для всех подписок Claude.
  • 🟣 Opus — самая умная, для серьёзной работы. Длинная стратегия, сложный анализ, тонкая правка большого документа.

Какую держать как основную:

  • Если у Вас подписка $100 Claude Max или выше — смело ставьте Opus и оставляйте надолго. Подписка покрывает её расход, а на Opus результаты заметно лучше на сложных задачах. Так использует, например, Дмитрий у себя — Opus всегда, под все задачи.
  • Если подписка $20 Pro — оставляйте Sonnet. На Pro квота на Opus быстро исчерпается, и Агент начнёт упираться в лимиты Anthropic. Sonnet справляется с 90% задач при гораздо большем запасе по квоте.
  • Haiku ставьте редко: только если делаете сотни мелких однотипных задач (типа массовой переформулировки) и хотите сберечь квоту для важного.

💬 Совет: под одну подписку выбирайте одну основную модель и работайте на ней. Постоянное переключение «на пробу» расходует Ваше внимание, а не результат.

Вкладка 2 — 🛠 Режим

Внутри две опции:

  • 👤 Обычный — Агент спрашивает перед действиями, которые могут быть рискованными (удаление, перезапись, отправка).
  • 🤖 Авто — Агент действует самостоятельно, без подтверждений.

Когда менять:

  • В первые недели работы — «Обычный». Вам важно видеть что Агент собирается сделать перед тем как он это сделает. Так быстрее учитесь.
  • Когда уже хорошо понимаете чего ожидать от Агента, и хочется меньше «лишних» подтверждений — переключайте на «Авто». Например, если каждый день генерируете отчёты по одному и тому же шаблону.

💬 Совет: для новичка — «Обычный» по умолчанию. Переходить на «Авто» имеет смысл когда Вы уже доверяете Агенту в конкретных типах задач.

Вкладка 3 — 🔑 Переменные окружения

Самая часто посещаемая вкладка. Здесь живут API-ключи внешних сервисов, которые Агент использует для своей работы.

В коробке заранее знакомы шесть сервисов:

  • 🎤 Deepgram (голосовые) — распознавание голоса. Без него голосовые сообщения не работают (подробности в статье «Голос: Deepgram + трюки»).
  • 🐙 GitHub (репозитории) — работа с кодом и репо. Агент сможет делать commit, push, pull, создавать PR.
  • 🗄️ Supabase (база данных) — для проектов, где нужна база данных в облаке.
  • ▲ Vercel (фронтенд) — деплой сайтов и Next.js-приложений.
  • 🧠 OpenRouter (модели AI) — единая «розетка» к десяткам других моделей: GPT, Gemini, DeepSeek, Llama, модели для генерации картинок и видео.

Это не замена Claude Code в Вашем Агенте — Агент по-прежнему думает на Claude. OpenRouter подключают, когда нужно дотянуться до другой модели по конкретной задаче. Примеры:

- Просите Агента сгенерировать картинку или видео прямо в чате (через FLUX, Stable Diffusion, Sora-подобные модели) — нужен ключ OpenRouter, чтобы Агент дёрнул нужную модель.

- Поручаете Агенту построить бота для клиента, в котором по требованиям клиента стоит GPT или Gemini — Ваш Агент использует OpenRouter, чтобы зашить туда нужную модель.

- Хотите сравнить ответы Claude и GPT на одну и ту же задачу, не заводя по отдельному ключу на каждый сервис.

  • 🔍 Voyage AI (семантический поиск) — поиск по смыслу во всей Вашей памяти (MEMORY.md, дневники, knowledge), а не только по точному совпадению слов.

Без него: спрашиваете «помнишь нашу переписку про продление хостинга», а в записях написано «оплата сервера на год» — Агент не найдёт, потому что слова разные.

С Voyage AI: тот же вопрос находит запись «оплата сервера на год», даже без совпадения по словам — потому что по смыслу это одно и то же. Ещё примеры:

- «Что мы решили про базу знаний» → найдёт запись «настроили Notion для команды».

- «Какие итоги встречи с Иваном» → найдёт «созвон с Иваном Петровичем».

- «Кто помогал с дизайном лендинга» → найдёт «отдал верстку Марине».

Возле подключённых сервисов будет зелёная галочка ✅. Нажмёте на сервис — Агент покажет короткую инструкцию где взять ключ (со ссылкой на нужную страницу), и попросит прислать его сообщением.

Безопасность ключей

Когда бот попросит ключ — Вы отправляете его обычным сообщением. Бот сразу удаляет Ваше сообщение с ключом и сохраняет значение в защищённое место на сервере (~/.agent/.env).

Никогда не отправляйте ключи в чат «просто так», без вызова /settings. В этом случае сообщение остаётся в истории Telegram, и ключ может утечь.

Кроме шести предустановленных, можно добавить любой свой ключ через кнопку «➕ Добавить переменную». Например, если используете сервис, которого нет в списке — даёте имя ключа и значение, Агент сохраняет.

Вкладка 4 — 🕐 Часовой пояс

Семь готовых вариантов:

  • UTC
  • Europe/Moscow
  • Europe/Kiev
  • Asia/Almaty
  • Asia/Vladivostok
  • America/New_York
  • America/Los_Angeles

Зачем настраивать: чтобы Агент понимал который у Вас час на самом деле. Иначе он будет думать в UTC и говорить «доброе утро» когда у Вас 11 вечера.

Часовой пояс влияет на:

  • Дневники в ~/workspace/memory/YYYY-MM-DD.md (Агент пишет дату по Вашему поясу).
  • Планы дня и расписание (если попросили план — он знает что «сейчас 9 утра», а не «21:00 UTC»).
  • Команду /status (показывает текущее время по Вашему поясу).

Когда менять: один раз в начале — выберите свой, и забудьте. Меняйте только если переехали в другой регион.

Вкладка 5 — 💰 Лимит

Дневной лимит расходов на запросы. Кнопка показывает сколько было бы потрачено сегодня против установленного лимита (например, «$12.40 / $50»).

При нажатии — выбор из пяти порогов: $10, $25, $50, $100, $200. По умолчанию — $50.

Главное про эту вкладку

Если Вы используете подписку Claude Pro или Max — эта вкладка по умолчанию не актуальна.

Агент работает через Вашу подписку Claude. Деньги за запросы Anthropic не списывает — Вы уже всё оплатили в рамках месячной подписки. Цифра «$X / $50» в этой вкладке — условная, она показывает «во сколько обошлись бы запросы, если бы они шли по API». Для подписки это просто индикатор активности, не реальный расход денег.

Вкладка становится по-настоящему важной только в одном случае: если Вы переключили Агента на работу через API (например, через свой Anthropic API-ключ или OpenRouter — оба настраиваются во вкладке «Переменные окружения»). Тогда там идут реальные доллары, и лимит работает как защитник от «разгона» бюджета.

Светофор расходов работает так:

  • 🟢 До 80% лимита — всё хорошо, никаких уведомлений.
  • 🟡 На 80% — Агент предупредит что Вы близко к лимиту.
  • 🔴 На 100% — новые запросы блокируются до завтра, чтобы не разогнаться больше чем планировали.

Когда менять:

  • Используете подписку — вкладку можно вообще не трогать. Стоит $50, и пусть стоит.
  • Используете API (свой ключ Anthropic или OpenRouter) — установите комфортный для бюджета лимит. Например, если ожидаете $5-10/день — поставьте $25, чтобы у Вас был запас и блок сработал раньше чем счёт уйдёт в небеса.
  • Лимит можно использовать как «защиту от runaway» — если Агент случайно зациклится в долгой задаче и за час нагенерирует тысячи запросов, блок остановит это сегодня.

Вкладка 6 — 🔄 Перезагрузка

Безопасно перезапускает бот: завершает текущий процесс, systemd сам поднимет его обратно через секунду. Контекст разговора, память, ключи — ничего не теряется.

Когда нажимать:

  • После того как руками что-то поправили в ~/.agent/.env через VS Code Tunnel — без перезагрузки бот не подхватит новые значения.
  • После обновления Конструктора, если Агент стал вести себя странно (обычно обновления применяются без рестарта, но иногда стоит).
  • Если бот стал «тормозить» после многочасовой работы — перезагрузка освобождает память.

Вкладка ✖ — Закрыть

Просто удаляет сообщение с меню. Никаких изменений не делает.

Мой Вам совет

В первый раз пройдитесь по меню сверху вниз, потратьте 5 минут:

  • Если у Вас подписка Max от $100 — поставьте модель Opus и оставьте надолго. Если Pro $20 — оставьте Sonnet.
  • Установите часовой пояс свой.
  • Лимит не трогайте, если используете подписку — это вкладка для тех, кто работает через API.
  • В переменных окружения подключите хотя бы Deepgram — без него голос не работает.
  • Режим оставьте на «Обычном».

Дальше месяцами не лазьте в /settings без причины. Когда нужно будет — например, добавить новый ключ или поднять лимит — откроете и сделаете точечно. Это меню сделано так, чтобы по умолчанию работало само.

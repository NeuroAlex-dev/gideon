Авторизация Telegram

В parser/.env не заполнены API_ID и API_HASH.

  1. Открой my.telegram.org → API development tools.
  2. Создай приложение.
  3. Скопируй API_ID и API_HASH в файл parser/.env.
  4. Перезапусти сервис: pm2 restart agent-parser.
  5. Обнови страницу.

Код пришёл в официальный чат Telegram.
